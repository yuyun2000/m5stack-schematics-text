# UiFlow2 Project Zone

> UiFlow2 版本的一大亮点是其全新的项目分享功能。这一功能不仅让每位用户都能将自己的独特创意和想法轻松地分享给开发者社区，同时也为大家提供了一个广阔的平台，让你可以随时随地浏览、下载并尝试其他开发者们精彩纷呈的项目。

## 功能预览

- My Project：历史编辑保存项目预览。
- Favorite Project：项目收集。
- Project Zone：项目分享。

## 1.My Project

点击 UiFlow2 Web IDE 右上角项目管理，查看我的最近编辑的历史项目。

<img src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow2/projects/project_zone/zone_01.png" width="70%">

用户每一次进入 Web IDE 时，系统默认会为用户创建一个新的项目，并在第一次运行时提醒用户保存该项目到项目目录中，如果用户需要修改该项目详细信息，如项目描述，项目代码，项目名称，删除项目等，都可以在 My Project 项目类中进行编辑。同时点击 Private 后，填写项目详细信息， 可发布到 Project Zone ，管理员审核成功后，项目会出现在 Project Zone 项目广场中，分享给其他开发者使用与下载。

<img src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow2/projects/project_zone/zone_02.png" width="70%">

## 2.Favorite Project

Favorite Project 类保存了在 Project Zone 中喜欢的其他用户分享的项目，为下次查找提供便利。

<img src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow2/projects/project_zone/zone_03.png" width="70%">

## 3.Project Zone

Project Zone 是一个专为编程爱好者打造的创意乐园！在这里，你可以尽情遨游于无尽的创意海洋中，找到各种设备的酷炫小项目，不仅可以一键获取灵感，还能轻松进行二次开发与使用。让你的编程之旅充满无限可能！

<img src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow2/projects/project_zone/zone_04.png" width="70%">

点击 import 按钮，WEB IDE 将会把该项目的编程步骤以及代码覆盖你当前的工作区。点击五角星保存到 Favorite Project 中方便下次使用。
