# UiFlow2 项目导入与导出

> UiFlow2 是一款强大的图形化编程工具，为各种硬件平台提供了直观易用的编程体验。为了方便用户在不同设备间共享和备份项目，UiFlow2 提供了文档的导入与导出功能。用户可以将项目导出为特定格式的文件，并在需要时从本地或其他设备导入，从而实现了项目的无缝迁移和版本管理。

## 导出到本地

使用 UiFlow2 进行项目导出的几个准备步骤：

- 1\. 移动鼠标至设备文件管理按钮上，点击 Export Project to local file
- 2\. 在浏览器中找到下载的导出文件，文档格式为 .m5f2

## 从本地导入

使用 UiFlow2 进行项目导出的几个准备步骤：

- 1\. 移动鼠标至设备文件管理按钮上，点击 Inport Project from local file
- 2\. 在资源管理器中找到需要导入的文件，文档格式必须为 .m5f2，导入后，代码块也会出现在界面

## 注意事项

- 版本兼容性：请确保你导出的文件与你的 UiFlow2 版本兼容。不同版本的 UiFlow 可能对文件格式或功能有所更改

- 文件安全：在导出和导入文件时，请确保你的文件没有被恶意软件篡改或损坏。建议使用可信的杀毒软件和存储设备

- 备份：在进行任何重要操作之前，请务必备份你的项目文件。这可以防止因意外情况导致的数据丢失
