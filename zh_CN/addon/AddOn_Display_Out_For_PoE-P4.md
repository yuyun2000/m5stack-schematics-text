## 描述

**AddOn Display Out For PoE-P4** 是一款适配 [Unit PoE-P4](/zh_CN/unit/Unit_PoE-P4) 的显示输出扩展模组。模块基于 `LT8912B` 芯片，可将 Unit PoE-P4 的 `MIPI DSI` 显示信号转换为 `HDMI` 输出，便于外接显示器实现本地图形界面、状态信息与控制画面的可视化呈现。

除显示输出外，模块还集成了触摸与本地存储相关扩展能力。板载 `TP` 接口用于连接触摸控制部分，提供触摸所需的 `I2C` 通信、复位与中断信号，便于实现触摸输入与界面交互；板载 microSD 插槽接入主控 `SDIO` 总线，可用于本地存储图片、界面资源、配置文件或其他显示相关数据，方便构建更完整的本地显示方案。模块同时引出 `USB Type-A` 接口，支持鼠标、键盘等外设接入扩展，适用于 PoE 显示终端、工业 HMI、信息看板与嵌入式控制面板等应用场景。

## 产品特性

- 适配 Unit PoE-P4 的显示输出扩展模组
- 基于 `LT8912B` 实现 `MIPI DSI` 转 `HDMI` 显示输出
- 标准 `HDMI Type-A` 接口，便于快速接入外部显示器
- 板载 `TP` 触摸接口，支持触摸输入与界面交互扩展
- 板载 microSD 插槽接入主控 `SDIO` 总线，支持本地存储图片、界面资源与配置文件
- 板载 `USB Type-A` 接口，支持鼠标、键盘等外设接入
- 支持构建显示、触摸、存储联动的一体化显示方案
- 支持`1280 x 720 @ 60Hz` 与 `1920 x 1080 @ 30Hz`两种输出分辨率

## 包装内容

- 1 x AddOn Display Out For PoE-P4
- 1 x 24P FPC 排线 (0.5mm 间距)

## 应用场景

- PoE 显示终端
- 工业 HMI
- 设备状态可视化看板
- 带触摸显示的人机交互控制面板
- 嵌入式设备本地显示与调试

### 24P FPC 输入接口

| 信号组      | 对应信号                                          |
| ----------- | ------------------------------------------------- |
| SDIO        | SDIO_CMD、SDIO_CK、SDIO_D0~D3                     |
| USB         | USB1_DP、USB1_DN                                  |
| TP (I2C)    | TP_RST、TP_INT、SYS_SDA、SYS_SCL                  |
| MIPI DSI    | MIPI_DSI_CK_P/N、MIPI_DSI_D0_P/N、MIPI_DSI_D1_P/N |
| 控制 / 电源 | RESET_N、3V3、5V、GND                             |

### 6P Touch 接口

| 接口     | 对应信号                                   |
| -------- | ------------------------------------------ |
| 6P Touch | 3V3、TP_RST、SYS_SCL、SYS_SDA、TP_INT、GND |
