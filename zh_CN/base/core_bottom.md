## 描述

**Base Bottom** 是一款 Core 基础型通用底座。它是 Basic 、 GRAY 底座的单独零售版，其硬件内容完全一致。内置锂电池，提供 M5-Bus 总线插座与侧面拓展排针排母，方便进行外接拓展。

## 产品特性

- 锂电池: 110 mAh @ 3.7V
