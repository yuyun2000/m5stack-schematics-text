# [Unit LoRa E220-433](/zh_CN/unit/Unit%20LoRaE220-433)

## 案例程序

注册 LoRaE220-433 并发送和接收数据

<img class="blockly_svg" src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow/blockly/unit/lora_e220_433/uiflow_block_example.svg">

```python
from m5stack import *
from m5ui import *
from uiflow import *
import unit

setScreenColor(0x404040)
lorae220_433_0 = unit.get(unit.LORA_E220_433, unit.PORTC)

lorae220_data1 = None
lorae220_rssi1 = None
rand = None

import random

def lorae220433_Zsicmcb(received_data, rssi):
  global lorae220_data1, lorae220_rssi1, rand
  try:
    lorae220_data1 = received_data.decode()
  except:
    lorae220_data1 = str(received_data)
  lorae220_rssi1 = rssi
  print((str('receive:') + str(lorae220_data1)))
  print((str('rssi:') + str(lorae220_rssi1)))
  pass

def buttonA_wasPressed():
  global lorae220_data1, lorae220_rssi1, rand
  rand = str(random.randint(1000000, 9999999))
  print((str('send random no:') + str(rand)))
  lorae220_433_0.send(0x0020, 0, ((str('0x0020:  ') + str(rand))))
  pass
btnA.wasPressed(buttonA_wasPressed)

def buttonC_wasPressed():
  global lorae220_data1, lorae220_rssi1, rand
  rand = str(random.randint(1000000, 9999999))
  print((str('send random no:') + str(rand)))
  lorae220_433_0.send(0x0025, 0, ((str('0x0025:  ') + str(rand))))
  pass
btnC.wasPressed(buttonC_wasPressed)

lorae220_433_0.uart_setup(1, 9600, None)
lorae220_433_0.enable_receive_callback(lorae220433_Zsicmcb)

```

## 功能说明

<img class="blockly_svg" src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow/blockly/unit/lora_e220_433/uiflow_block_unit_lorae220433_available_data.svg">

```python
print((str('available of data:') + str((lorae220_433_0.available_data()))))
```

- 返回一个整数，用于计算在不阻塞的情况下可以读取的字节数。 如果没有可用字节，它将返回 0，如果有字节，则返回正数。

<img class="blockly_svg" src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow/blockly/unit/lora_e220_433/uiflow_block_unit_lorae220433_get_setup_parameter.svg">

```python
print((str('address:') + str((lorae220_433_0.read_configure_param()[0]))))
```

- 获取LoRa E200设备的设备地址参数

<img class="blockly_svg" src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow/blockly/unit/lora_e220_433/uiflow_block_unit_lorae220433_receive_data_timeout.svg">

```python
print((str('receive data:') + str((lorae220_433_0.receive(1000)))))
```

- 使用阻塞方式接收数据。 用于设置接收超时，单位为 ms

<img class="blockly_svg" src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow/blockly/unit/lora_e220_433/uiflow_block_unit_lorae220433_receive_in_loop.svg">

```python
lorae220_433_0.enable_receive_callback(lorae220433_nFWKgcb)
```

- 使用非阻塞模式接收数据

<img class="blockly_svg" src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow/blockly/unit/lora_e220_433/uiflow_block_unit_lorae220433_send_list.svg">

```python
lorae220_433_0.send(0x0000, 0, [0x10, 0x20])
```

- 向指定的目标地址和频道发送数据(send raw data)

<img class="blockly_svg" src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow/blockly/unit/lora_e220_433/uiflow_block_unit_lorae220433_send_str.svg">

```python
lorae220_433_0.send(0x0000, 0, 1000)
```

- 向指定的目标地址和频道发送数据

<img class="blockly_svg" src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow/blockly/unit/lora_e220_433/uiflow_block_unit_lorae220433_setup.svg">

```python
lorae220_433_0.setup(0x0000, 0, encryption_key=0x0000)
```

- 设置模块参数
  - address：本地地址
  - channel：本机频道
  - encryption_key：加密密钥

<img class="blockly_svg" src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow/blockly/unit/lora_e220_433/uiflow_block_unit_lorae220433_setup_uart_id.svg">

```python
lorae220_433_0.uart_setup(1, 9600, None)
```

- 设置 UART ID /波特率/奇偶校验

<img class="blockly_svg" src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow/blockly/unit/lora_e220_433/uiflow_block_unit_lorae220433_stop_receive.svg">

```python
lorae220_433_0.disable_receive_callback()
```

- 停止在非阻塞模式下接收数据
