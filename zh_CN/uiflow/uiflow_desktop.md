# UIFlow-Desktop-IDE

该版本为 UiFlow1.x 的离线版本， 方便在一些无网络的环境中， 通过 USB 连接的方式对设备进行编程。

## 1. 软件下载

点击下方对应操作系统的`UIFlow-Desktop-IDE`进行下载。

| 操作系统 | 下载链接                                                                                                |
| -------- | ------------------------------------------------------------------------------------------------------- |
| Windows  | [Download](https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/software/UIFlow-Desktop-IDE.zip)       |
| MacOS    | [Download](https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/software/UIFlow-Desktop-IDE_MacOS.zip) |
| Linux    | [Download](https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/software/UIFlow-Desktop-IDE_Linux.zip) |

## 2.USB 编程模式

1\. 将下载好的 UIFlow Desktop IDE 压缩包解压，双击执行应用程序。

<img src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow/uiflow_desktop/uiflow_desktop_01.jpg" width="70%">

2\. 软件启动后，将自动检测电脑是否安装有 USB 驱动（CP210X）。点击 `Install`，根据提示进行安装。

3\. 驱动安装完成后，将自动进入 UIFlow Desktop IDE 并自动弹出配置框，此时将 M5 设备通过 type-C 数据线连接至电脑。

\#> 注意 | 使用 UIFlow Desktop IDE 需要 M5 设备搭载 UiFlow 固件且进入 USB 编程模式。<br/><img src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow/uiflow_desktop/uiflow_desktop_02.jpg" width="70%"><br/>以 M5Core 为例，单击设备左侧电源键重启，进入菜单后快速选择`Setup`，进入配置页面后，选择`USB mode`。<br/><img src="https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/docs/static/assets/img/uiflow/uiflow_desktop/uiflow_desktop_03.jpg" width="70%">

4\. 选择好对应的端口与编程设备，点击`OK`进行连接。
