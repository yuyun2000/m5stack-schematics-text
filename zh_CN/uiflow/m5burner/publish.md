# Publish Firmware

\#> 功能说明 | 本教程将向你介绍如何通过`M5Burner`上传和`分享`自己的`固件程序`。注：使用该功能前，请点击右上角登录按钮， 使用 M5Stack 社区账号进行登录。

1. 打开 M5Burner 软件后， 点击左下角的`USER CUSTOM`, 选择`Publish`选项。

<img src="https://static-cdn.m5stack.com/resource/docs/static/assets/img/uiflow/m5burner/m5burner_publish_02.png" width="70%">

2. 填写发布固件的相关信息并单击`Upload`。

<img src="https://static-cdn.m5stack.com/resource/docs/static/assets/img/uiflow/m5burner/m5burner_publish_01.png" width="70%">

\#> 填入信息说明 | `Name`: 所分享固件的名称 <br/>`Version`: 该固件的版本号 <br/>`Description`: 该固件的相关简介及使用说明<br/>`Device Type`: 该固件的主机设备类型 <br/>`Github`: 该固件程序的 Github 网页链接<br/>`FirmWare`: 所分享的固件（推荐使用 M5Burner 附带的[firmware Export 功能](/zh_CN/uiflow/m5burner/publish)）<br/>`Cover`: 在 Burner 显示该固件的封面配图

3. 等待固件上传成功， 若固件上传成功则会在当前界面看到已上传的固件。

<img src="https://static-cdn.m5stack.com/resource/docs/static/assets/img/uiflow/m5burner/m5burner_publish_03.png" width="70%">

- 点击`Detail`按钮可以对固件的相关信息进行二次修改

- 点击`Publish`按钮可以更改固件的公开状态

- 点击`Share`按钮可以获取该固件的`Share Code`, 可以将`Share Code`分享给其他人，并使用`Share Burn`功能烧录该分享固件

- 点击`Remove`按钮可以从 Burner 中删除该固件
