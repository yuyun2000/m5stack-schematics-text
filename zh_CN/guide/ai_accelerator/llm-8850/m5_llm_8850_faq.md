# 常见问题

## 找不到设备

错误代码 **0x8030010b**

```bash
m5stack@raspberrypi:~ $ axcl-smi
[08-11 16:01:16:933][E][smi][device_manager][  20]: axcl init fail, ret = 0x8030010b
[2025-08-11 16:01:16.933][2110][E][device manager][init][39]: open /dev/axcl_host fail, errno: 2 No such file or directory
+------------------------------------------------------------------------------------------------+
| AXCL-SMI  V3.6.3_20250722020142                                  Driver  V3.6.3_20250722020142 |
+-----------------------------------------+--------------+---------------------------------------+
| Card  Name                     Firmware                                                          | Bus-Id       | Memory-Usage |
| Fan   Temp                Pwr:Usage/Cap                                                          | CPU      NPU | CMM-Usage    |
| =========================================+==============+======================================= |

+------------------------------------------------------------------------------------------------+
| Processes:                                                                                       |
| Card      PID  Process Name                                                   NPU Memory Usage   |
| ================================================================================================ |
```

排查方法：

- 使用 **ls pci** 检查是否有 **Multimedia video controller: Axera Semiconductor Co., Ltd Device 0650 (rev 01)** 设备。
- 检查 LLM-8850 Card 是否插好，断开树莓派电源重启。
- 卸载并重新安装驱动。
