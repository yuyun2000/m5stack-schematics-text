# [Silero-vad](https://github.com/snakers4/silero-vad)

## 介绍

Silero VAD（语音活动检测）是一种用于检测音频流中是否有人声的模型。它适用于需要区分语音和非语音片段的应用场景，比如语音助手、转录服务以及音频处理任务。

## 可用的 CPU 模型

### llm-model-silero-vad

- 支持多语言语音检测，包括英语、中文和日语。  
- 支持平台：LLM630 计算套件、Module LLM 和 Module LLM Kit 及 AI Pyramid

### 安装

```shell
apt install llm-model-silero-vad
```
