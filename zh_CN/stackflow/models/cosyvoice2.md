# [CosyVoice2-0.5B](https://huggingface.co/FunAudioLLM/CosyVoice2-0.5B)

## 介绍

CosyVoice2-0.5B 是由 FunAudioLLM 提供的高质量多语言带音色克隆的文本转语音模型。

## 可用的 NPU 模型

### cosyvoice2-0.5b-ax650

- 支持语言：中，英，日，韩等
- 支持平台：AI Pyramid
- 单次最长输出音频：27s 
- ttft：401.84ms
- RTF：1.36

### 安装

```shell
apt install llm-model-cosyvoice2-0.5b-ax650
```
