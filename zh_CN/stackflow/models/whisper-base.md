# [Whisper-base](https://huggingface.co/openai/whisper-base)

## 介绍

Whisper 是一个预训练的自动语音识别（ASR）和语音翻译模型。该模型基于680,000小时标注数据训练，展示了在无需微调的情况下对多种数据集和领域的强大泛化能力。

## 可用的 NPU 模型

### whisper-base

#### **支持平台**：LLM630 Compute Kit、Module LLM 和 Module LLM Kit

- 该模型支持多语言语音识别和翻译。

- 编码时间：660.31 毫秒

- 平均解码时间：51.11 毫秒
