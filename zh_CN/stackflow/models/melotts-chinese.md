# [MeloTTS-Chinese](https://huggingface.co/myshell-ai/MeloTTS-Chinese)

## 介绍

MeloTTS 是由 MyShell.ai 提供的高质量多语言文本转语音库。

- 中文语音支持中英文混合。

## 可用的 NPU 模型

### 中文混合英文模型

#### melotts-zh-cn

**支持平台**：LLM630 Compute Kit、Module LLM 和 Module LLM Kit

### 安装

```shell
apt install llm-model-melotts-zh-cn
```

- [下载 llm-model-melotts-zh-cn](https://repo.llm.m5stack.com/m5stack-apt-repo/pool/jammy/ax630c/v0.6/llm-model-melotts-zh-cn_0.6-m5stack1_arm64.deb)

**支持平台**： Raspberry Pi5

### 安装

```shell
sudo apt install llm-model-melotts-zh-cn-axcl
```

- [下载 llm-model-melotts-zh-cn-axcl](https://repo.llm.m5stack.com/m5stack-apt-repo/pool/bookworm/llm8850/v0.6/llm-model-melotts-zh-cn-axcl_0.6-m5stack1_arm64.deb)

**支持平台**： AI Pyramid

### 安装

```shell
apt install llm-model-melotts-zh-cn-ax650
```

- [下载 llm-model-melotts-zh-cn-axcl](https://repo.llm.m5stack.com/m5stack-apt-repo/pool/jammy/ax650c/v0.7/llm-model-melotts-zh-cn-ax650_0.7-m5stack1_arm64.deb)
