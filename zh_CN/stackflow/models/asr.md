# [自动语音识别（Automatic Speech Recognition）](https://k2-fsa.github.io/sherpa/ncnn/pretrained_models/zipformer-transucer-models.html)

## 介绍

自动语音识别（ASR）模型旨在将口语转换为文本。这些模型可广泛应用于转录服务、语音控制系统和辅助工具等场景中。

## 可用的 CPU 模型

### llm-model-sherpa-ncnn-streaming-zipformer-20m-2023-02-17

**支持平台**：LLM630 Compute Kit、Module LLM 和 Module LLM Kit 及 AI Pyramid

- 该模型是基于 Zipformer 架构的流式 ASR 模型，使用大型数据集训练，能够提供高精度的语音识别能力。

- 此模型**仅支持英文**。

#### 安装

```shell
apt install llm-model-sherpa-ncnn-streaming-zipformer-20m-2023-02-17
```
