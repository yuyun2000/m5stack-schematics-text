# [Yolo11](https://github.com/ultralytics/ultralytics)

## 介绍

YOLOv11 是一种常用的目标检测模型，基于 YOLO（You Only Look Once）架构构建。它设计目标是速度快且高效，同时保持对图像和视频中物体的高准确检测能力。

## 可用的 NPU 模型

### yolo11n

**支持平台**：LLM630 计算套件、Module LLM 和 Module LLM 套件

- 该模型支持高准确度和高速的目标检测任务。

#### 安装

```shell
apt install llm-model-yolo11n
```

- [下载 llm-model-yolo11n](https://repo.llm.m5stack.com/m5stack-apt-repo/pool/jammy/ax630c/v0.2/llm-model-yolo11n_0.2-m5stack1_arm64.deb)
