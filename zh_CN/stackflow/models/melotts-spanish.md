# [MeloTTS-Spanish](https://huggingface.co/myshell-ai/MeloTTS-Spanish)

## 介绍

MeloTTS 是 MyShell.ai 推出的高质量多语言文本转语音（TTS）库，支持自然流畅的语音合成。

## 可用 NPU 模型

### 西班牙语模型

#### melotts-es-es

- 支持平台：LLM630 计算套件、Module LLM 和 Module LLM 套件

### 安装

```bash
apt install llm-model-melotts-es-es
```

- [下载 llm-model-melotts-es-es](https://repo.llm.m5stack.com/m5stack-apt-repo/pool/jammy/ax630c/v0.5/llm-model-melotts-es-es_0.5-m5stack1_arm64.deb)

**支持平台**： Raspberry Pi5
