# [关键词识别（Keyword Spotting）](https://k2-fsa.github.io/sherpa/onnx/kws/index.html)

## 介绍

关键词识别（KWS）模型用于在音频流中检测特定的关键词。这类模型在语音助手、智能家居设备以及其他需要识别特定命令或短语的应用场景中具有重要作用。

## 可用的 CPU 模型

### llm-model-sherpa-onnx-kws-zipformer-gigaspeech-3.3m-2024-01-01

**支持平台**：LLM630 Compute Kit、Module LLM、Module LLM Kit 及 AI Pyramid

- 该模型基于 GigaSpeech 数据集训练，用于关键词识别任务。
- 该模型**仅支持英语**。

#### 安装

```shell
apt install llm-model-sherpa-onnx-kws-zipformer-gigaspeech-3.3m-2024-01-01
```
