# [MeloTTS-English](https://huggingface.co/myshell-ai/MeloTTS-English)

## 介绍

MeloTTS 是由 MIT 和 MyShell.ai 联合打造的高质量多语言文本转语音库。

- 英文语音支持美式、英式、印式、澳洲及默认口音。

## 可用的 NPU 模型

### 默认发音人模型

#### melotts-en-default

**支持平台**：LLM630 Compute Kit、Module LLM 和 Module LLM Kit

#### 安装

```shell
apt install llm-model-melotts-en-default
```

- [下载 llm-model-melotts-en-default](https://repo.llm.m5stack.com/m5stack-apt-repo/pool/jammy/ax630c/v0.6/llm-model-melotts-en-default_0.6-m5stack1_arm64.deb)

**支持平台**： Raspberry Pi5

### 安装

```shell
sudo apt install llm-model-melotts-en-default-axcl
```

- [下载 llm-model-melotts-en-default-axcl](https://repo.llm.m5stack.com/m5stack-apt-repo/pool/bookworm/llm8850/v0.6/llm-model-melotts-en-default-axcl_0.6-m5stack1_arm64.deb)
