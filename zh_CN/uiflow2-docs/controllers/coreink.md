###### UI


![op.png](../../_static/controllers/coreink/coreink_op.png)

CoreInk has 4 buttons, **Up**, **Down**, **Center**, and **Btn.EXT**.

![flow.png](../../_static/controllers/coreink/coreink_flow.png)

**Btn.EXT** key cycles through the available applications.

![config.png](../../_static/controllers/coreink/coreink_config.png)

**Down** key toggles between available servers.

**Btn.EXT** key cycles through the available applications.

![applist.png](../../_static/controllers/coreink/coreink_applist.png)

**Up** and **Down** keys cycle through scripts.

**Center** key runs the specified script.

**Btn.EXT** key cycles through the available applications.
