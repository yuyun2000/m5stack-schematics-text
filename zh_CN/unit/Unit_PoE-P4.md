## 描述

**Unit PoE‑P4** 是一款高性能 PoE 以太网嵌入式控制器，核心搭载 **ESP32‑P4** 主控（RISC‑V 32 位双核 360MHz + LP 单核 40MHz），集成 **16MB Flash** 与 **32MB PSRAM**，内置 **IP101GRI** 以太网 PHY 芯片，支持 **10/100 Mbps** 通信速率，同时集成 PoE 供电，最大功率 6W，内置 DC‑DC 转换为整机供电，有效简化应用电源布线；设备配备 24P FPC 显示扩展接口（MIPI DSI + TP + BL）与 24P FPC 摄像头接口（MIPI CSI），支持高分辨率屏幕显示与图像采集，扩展方面提供 1 个 USB Type‑C Host（USB 2.0）接口用于连接 U 盘、HID 人机交互设备等外设，1 个支持 OTG 与固件下载功能的 USB Type‑C（USB 2.0）接口用于调试，同时配有 Hat 扩展接口、SDIO & ISP IO 扩展接口及 Grove 接口，可灵活接入各类传感器与外设，设备还集成 RGB 指示灯、红外发射电路与用户按键，满足状态指示、本地交互与红外发射信号需求，适用于工业 HMI、PoE 智能终端、视觉采集节点、智能家居中控与边缘计算设备等场景。

## 产品特性

- ESP32-P4 核心主控：
  - 16MB Flash + 32MB PSRAM
- 外设接口：
  - 1x 24P FPC 摄像头接口： MIPI CSI 2‑Lane
  - 1x 24P FPC 显示扩展接口： MIPI DSI 2‑Lane + TP + BL
  - 1x USB Type‑C Host 接口 (USB 2.0)
  - 1x USB Type‑C 程序下载 / OTG 接口 (USB 2.0)
- 以太网：
  - 集成 IP101 10/100M 以太网 PHY 芯片
  - 支持 PoE（有源以太网）供电，通过单根网线同时实现数据传输与供电
- 人机交互：
  - 1x RGB LED 指示灯
  - 1x 用户按键
  - 1x 红外发射
- 拓展接口：
  - HY2.0-4P 拓展接口
  - Hat2-Bus 2.54-16P
  - SDIO-Bus 2.54-9P
  - ISP-Bus 2.54-6P

## 操作说明

### 进入下载模式

如需烧录固件，请将设备通过 USB Type-C 数据线连接至电脑，长按侧面的按键 3 秒直到绿灯亮起，此时设备进入下载模式，等待烧录。

<img src="https://m5stack-doc.oss-cn-shenzhen.aliyuncs.com/1217/U213-download-mode.gif" width="40%">

### 访问设备控制页面

设备出厂固件默认预置 WEB 控制页面，支持 RGB LED 颜色控制、按钮计数统计、GPIO 引脚模式切换与高低电平输出，可按以下步骤访问设备控制页面：

1. Unit PoE-P4 供电并接入以太网。
2. 打开串口监视器，确认设备当前 IP 地址。
3. 确保你的电脑或手机与设备处于同一局域网（LAN）。
4. 使用以下任一方式访问页面：

<img src="https://m5stack-doc.oss-cn-shenzhen.aliyuncs.com/1217/unit_poe_p4_userdemo_03.jpg" width="50%">

- 在浏览器输入 `poe-p4.local`
- 将串口监视器中显示的 IP 地址复制到浏览器地址栏访问

<img src="https://m5stack-doc.oss-cn-shenzhen.aliyuncs.com/1217/unit_poe_p4_userdemo_03.png" width="50%">
