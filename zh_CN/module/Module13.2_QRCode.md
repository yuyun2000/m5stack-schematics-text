## 描述

**Module13.2 QRCode** 是 M5Stack 堆叠模块系列中的一款一维 / 二维码扫描模块，内部集成了 640 x 480 分辨率的 CMOS 二维码采集引擎，支持市场上主流的 3 类二维码和 14 类一维码，能够快速、精准地识别印刷品、标签等介质上的条码信息。通过拨动开关可实现 QR 模组 USB 和 UART 通信接口的切换，用于外部设备与 QR 模组通信，其中 USB 接口可配置为 USB 虚拟串口、USB HID 键盘、USB HID-POS 三种模式。模块内置蜂鸣器音效提示与照明、定位 LED，提供丰富的二次开发接口，支持多种扫码触发模式（如自动模式、连续模式、运动感应模式、脉冲模式等）进行功能定制和集成开发。模块集成 DC 电源输入接口， 支持 DC 9 ~ 24V 输入供电，经由内部 DC-DC 降压电路可为整个设备提供电源支持，适用于物流仓储、零售收银、票务系统、生产制造追溯、医疗信息采集、自助终端等多种场景。

## 产品特性

- 支持 3 类二维码和 14 类一维码
- 内置 USB 和 UART 通讯接口切换开关
- 内置蜂鸣器音效提示
- 内置照明 LED
- 高分辨率成像
- 对焦与瞄准功能
- 多种扫码触发模式，自动、连续模式等
- DC 接口供电，宽电压输入：DC 9 ~ 24V
- 开发平台:
  - Arduino
  - UiFlow2

## 管脚映射

### IO 拓展芯片

| PI4IOE5V6408 | P0       | P4   |
| ------------ | -------- | ---- |
| QR Module    | QR_5V_EN | TRIG |

### M5-Bus

\#> Switch | 下方 M5-Bus 中标记 `SW` 的引脚，表示可通过拨码开关进行切换，用于适配不同的主控设备；如果切换到`NC`则表示不连接该引脚。

::m5-bus-table
| PIN                         | LEFT | RIGHT | PIN                         |
| --------------------------- | ---- | ----- | --------------------------- |
| GND                         | 1    | 2     |                             |
| GND                         | 3    | 4     | PORT.B                      |
| GND                         | 5    | 6     |                             |
|                             | 7    | 8     |                             |
|                             | 9    | 10    | PORT.B                      |
|                             | 11   | 12    |                             |
| QR_TX / NC (SW)             | 13   | 14    | QR_RX / NC (SW)             |
| QR_TX / PORT.C_UART_RX (SW) | 15   | 16    | QR_RX / PORT.C_UART_TX (SW) |
| I2C_SDA                     | 17   | 18    | I2C_SCL                     |
|                             | 19   | 20    |                             |
| QR_RX / NC (SW)             | 21   | 22    | QR_TX / NC (SW)             |
| QR_RX / NC (SW)             | 23   | 24    |                             |
| HPWR                        | 25   | 26    | QR_TX / NC (SW)             |
| HPWR                        | 27   | 28    | 5V                          |
| HPWR                        | 29   | 30    |                             |
::

### HY2.0-4P

::grove-table
| HY2.0-4P | Black | Red | Yellow                  | White                   |

| PORT.B   | GND   | 5V  | OUT                     | IN                      |
| PORT.C   | GND   | 5V  | UART_TX/QR_USB_DP+ (SW) | UART_RX/QR_USB_DM- (SW) |
