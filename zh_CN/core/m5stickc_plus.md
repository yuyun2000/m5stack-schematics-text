## 描述

**StickC-Plus** 是[M5StickC](/zh_CN/core/m5stickc)的大屏幕版本。其主控采用 ESP32-PICO-D4 模组，具备 Wi-Fi 功能。在小巧的机身内部，集成了丰富的硬件资源，比如红外、RTC、麦克风、LED、IMU、按键、蜂鸣器、PMU 等。它在保留原有 M5StickC 功能的基础上，新增了无源蜂鸣器。同时，屏幕尺寸升级到了 1.14 寸，分辨率为 135 x 240 的 TFT 屏幕，相比之前 0.96 寸的屏幕，显示面积增加了 18.7%。电池容量达到了 120mAh ，接口同样支持 HAT 与 Unit 系列产品。

这款小巧精致的开发工具，能够激发出无限的创作潜力。StickC-Plus 可以助力快速搭建物联网产品原型，极大地简化整个开发流程。即便对于刚刚开始接触编程开发的初学者而言，也能够借助它搭建出一些有趣的应用，并将其应用到实际生活当中。

## 产品特性

- 基于 ESP32 开发，支持 Wi-Fi
- 内置 3 轴加速计与 3 轴陀螺仪
- 内置 Red LED
- 集成红外发射管
- 内置 RTC
- 集成麦克风
- 用户按键，LCD (1.14 寸)，电源 / 复位按键
- 120 mAh 锂电池
- 拓展接口
- 集成无源蜂鸣器
- 可穿戴 & 可固定
- 开发平台
  - UiFlow1
  - UiFlow2
  - Arduino IDE
  - ESP-IDF
  - PlatformIO

- [StickC-Plus 恢复出厂固件教程](/zh_CN/guide/restore_factory/m5stickc_plus)

**注意：**

- StickC-Plus 支持的波特率： 1200 ~115200，250K，500K，750K，1500K

- G36/G25 共用同一个端口，当使用其中一个引脚时要将另外一个引脚设置为浮空输入

- 比如要使用 G36 引脚作为 ADC 输入，则配置 G25 引脚为浮空状态

- VBUS_VIN 与 VBUS_USB 的输入范围限制在 4.8-5.5V，VBUS 供电时将通过 AXP192 电源管理为内置电池进行充电。

```cpp
setup()
{
   M5.begin();
   pinMode(36，INPUT);
   gpio_pulldown_dis(GPIO_NUM_25);
   gpio_pullup_dis(GPIO_NUM_25);
}
```
