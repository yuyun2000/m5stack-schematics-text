## 描述

Xiaozhi Card（小智墨伴）是一款基于 ESP32-S3 的智能语音终端。它支持 4G 和 Wi-Fi 网络连接，配备 2.7 英寸触控电子墨水屏，搭配麦克风和扬声器，带来流畅易用的语音交互体验。内置 300mAh 电池，支持充电管理和电量检测，实现长时间待机与便捷充电。

Xiaozhi Card 具备按钮、触控、语音三种唤醒方式，满足不同操作习惯；采用磁吸设计，可轻松吸附在底座或冰箱等金属表面。Pogo Pin 磁吸触点实现主机与底座的通信和充电连接，底座提供 USB Type-C 接口、Grove 扩展口、复位和烧录按键，以及 RGB 指示灯。产品开箱即用，配置简便，是语音助手、数字助理、MCP 终端等智能交互场景的理想选择。

## 产品特性

- Xiaozhi Card（小智墨伴）主机

  - ESP32-S3-PICO 主控，支持 Wi-Fi 连接
  - 2.7 英寸触控电子墨水屏，分辨率 176 x 264
  - 4G 网络（LTE Cat.1），支持中国移动 / 中国联通 / 中国电信的 nano SIM 卡
  - 电池容量 300mAh，支持充放电管理，检测剩余电量和充电状态
  - 麦克风、扬声器（支持音量调节）
  - 内置 64MB SD NAND
  - 支持通过背部按钮、触控、语音唤醒
  - 开箱即用，配置简单
  - 磁吸设计，可吸附至底座或冰箱等金属表面
  - Pogo Pin 磁吸触点，连接底座通信和充电
  - 预留挂绳孔、背部 M2 螺丝孔，适用多种场景

- Xiaozhi Card Base 底座
  - USB Type-C 接口，用于通信和供电
  - HY2.0-4P Grove 接口
  - 复位 / 烧录按钮
  - RGB 指示灯
  - Pogo Pin 磁吸触点，连接主机通信和充电
  - 乐高兼容孔，方便固定

## 操作说明

### SIM 卡

产品附带一张中国移动的物联网卡，包含每月 1GB 的小智定向流量，有效期一年，从首次入网时间开始计算。有效期过后也可以前往 http://iotwe.yshiqi.com/recharge.php 续费充值。<br>如果 SIM 卡在一年体验期内，**充值管理页面**会显示 “此卡片不符合线上充值条件”（如下图所示），但是不影响正常使用。建议到期后再进行充值。

<img src="https://m5stack-doc.oss-cn-shenzhen.aliyuncs.com/1157/xiaozhi_card_sim_card_02.jpg" width="50%">

!> 附带的 SIM 卡仅限本产品使用 | 设备出厂时默认插入了配套的物联网卡（nano SIM），请勿将该 SIM 卡用于其他设备、其他用途，否则会被锁卡。如果使用 4G 网络，则需要安装 nano 尺寸的 SIM 卡，安装方向见下图。若安装错误请用镊子等工具小心取出，避免损坏设备及 SIM 卡。支持 SIM 卡热插拔，插卡后不需要重启，不过 4G 通信从冷启动到建立连接需要一定时间，可能在 20 秒左右，根据运营商信号情况有所不同。

<img src="https://m5stack-doc.oss-cn-shenzhen.aliyuncs.com/1157/xiaozhi_card_sim_card_01.jpg" width="50%">

\#> SIM 卡兼容性 | 除附带的 SIM 卡以外，也可以使用其他卡。<br> **支持的运营商**：<br> 中国移动、中国联通、中国电信<br> **支持的频段**：<br> LTE-FDD Band 1 / 3 / 5 / 8<br>LTE-TDD Band 34 / 38 / 39 / 40 / 41

### 按键操作

- **开机**：**长按两秒** Xiaozhi Card 背面的圆形按键。

- **关机**：**双击（连续短按两次）** Xiaozhi Card 背面的圆形按键。开机状态下**从上到下**滑动屏幕，也可以调出关机选项。

- **待机 / 唤醒**：开机状态下单击背部圆形按键切换待机（待命）与唤醒（聆听）状态。

- **复位**：在 Xiaozhi Card 与 Base 连接并通过 USB 接通电源时，**短按** Xiaozhi Card Base 上的按键，重启设备。

- **烧录**：在 Xiaozhi Card 与 Base 连接并通过 USB 接通电源时，**长按** Xiaozhi Card Base 上的按键，直至绿灯亮起，进入下载模式。

### Base RGB 灯状态

- <span style="color:green">**绿色**</span>：聆听中

- <span style="color:blue">**蓝色**</span>：充电中

\#> 充电说明 | 设备充电时，设备将保持开启，无法进行关机。

<img src="https://m5stack-doc.oss-cn-shenzhen.aliyuncs.com/1157/xiaozhi_card_base_light_01.jpg" width="80%" />

### 设备复位

?> 复位说明 | 当 Xiaozhi Card 出现无响应的情况时，请将其连接到已经通电的充电底座上，然后按底座的复位按钮进行重启。
