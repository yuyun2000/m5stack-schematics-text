## 描述

PaperColor 是一款配备 4 英寸 E Ink Spectra 6 全彩墨水屏的开发板，屏幕分辨率为 400x600，兼具低功耗与高可视性优势。内置集成 ESP32-S3R8 核心主控，配备 16MB Flash 与 8MB PSRAM，支持 2.4GHz Wi-Fi 无线通信。开发板集成了完整的人机交互系统，包括 3 个可编程按键、基于 ES8311 音频编解码器的音频系统、具备 AEC 回声消除功能的 MEMS 麦克风以及 AW8737A 功放驱动的扬声器，支持高质量语音拾取与音频播放。扩展功能方面，板载 SHT40 温湿度传感器、RX8130CE 实时时钟、microSD 卡槽、红外发射管、2 个 RGB LED 及 HY2.0-4P 扩展接口，配合 M5PM1 多级电源管理系统与 1250mAh 锂电池，可实现持久的低功耗运行。适用于电子标牌、语音交互终端、环境监测仪表、艺术展示装置等物联网与嵌入式场景。

## 产品特性

- ESP32-S3R8 核心主控：
  - 16MB Flash + 8MB PSRAM
  - 2.4GHz Wi-Fi
- 4 英寸 E Ink Spectra 6 全彩墨水屏
- 内置 1250mAh 锂电池
- microSD 卡槽
- 音频交互:
  - ES8311 音频编解码器
  - 1W 扬声器 + MEMS 麦克风
- 人机交互:
  - 3x 用户按键 + 1x 电源按键
  - 红外发射管
  - 2x RGB LED
- 板载 SHT40 温湿度传感器
- RX8130CE 实时时钟

## 管脚映射

### E-Paper

| ESP32S3R8 | G15     | G13      | G44     | G43     | G11       | G12      |
| --------- | ------- | -------- | ------- | ------- | --------- | -------- |
| EL040EF1  | SPI_CLK | SPI_MOSI | EINK_CS | EINK_DC | EINK_BUSY | EINK_RST |

| M5PM1    | PYG0      |
| -------- | --------- |
| EL040EF1 | PY_EPD_EN |

### KEY

| ESP32S3R8 | G1                   | G9                   | G10                  |
| --------- | -------------------- | -------------------- | -------------------- |
| KEY       | USER_KEY1 (Button C) | USER_KEY2 (Button B) | USER_KEY3 (Button A) |

### IR

| ESP32S3R8 | G48   |
| --------- | ----- |
| IR        | IR_TX |

### RGB

| ESP32S3R8 | G21 |
| --------- | --- |
| RGB LED   | RGB |

### Audio

| ESP32S3R8     | G2            | G3            |
| ------------- | ------------- | ------------- |
| ES8311 (0x18) | AUDIO_I2C_SCL | AUDIO_I2C_SDA |
| ES7210 (0x40) | AUDIO_I2C_SCL | AUDIO_I2C_SDA |

| ESP32S3R8 | G42      | G41      | G40      | G39       | G38       |
| --------- | -------- | -------- | -------- | --------- | --------- |
| ES8311    | I2S_MCLK | I2S_LRCK | I2S_BCLK |           | I2S_DSDIN |
| ES7210    | I2S_MCLK | I2S_LRCK | I2S_BCLK | I2S_SDOUT |           |

| ESP32S3R8                             | G45          |
| ------------------------------------- | ------------ |
| ES8311 & ES7210 Power (CODEC_3V3_L3B) | AUDIO_PWR_EN |

| ESP32S3R8 | G46    |
| --------- | ------ |
| AW8737A   | SPK_EN |

- G46(SPK_EN): 扬声器功放使能
- G45(AUDIO_PWR_EN): 音频编解码芯片与麦克风供电

### RTC

| ESP32S3R8       | G7      | G3      | G2      |
| --------------- | ------- | ------- | ------- |
| RX8130CE (0x32) | RTC_IRQ | SYS_SDA | SYS_SCL |

### SHT40

| ESP32S3R8    | G3      | G2      |
| ------------ | ------- | ------- |
| SHT40 (0x44) | SYS_SDA | SYS_SCL |

### microSD

| ESP32S3R8 | G47 | G15     | G13      | G14      |
| --------- | --- | ------- | -------- | -------- |
| microSD   | CS  | SPI_CLK | SPI_MOSI | SPI_MISO |

### HY2.0-4P

::grove-table
| HY2.0-4P | Black | Red | Yellow | White |
| -------- | ----- | --- | ------ | ----- |
| PORT.A   | GND   | 5V  | G4     | G5    |
::

### M5PM1

| ESP32S3R8    | G3      | G2      |
| ------------ | ------- | ------- |
| M5PM1 (0x6e) | SYS_SDA | SYS_SCL |

| M5PM1   | DCDC3V3_EN_PP | LDO3V3_EN_PP  | BOOST5V_EN_PP   |
| ------- | ------------- | ------------- | --------------- |
| 3V3_L2  | PY_MPWR_EN    |               |                 |
| RGB LED |               | PY_RGB_PWR_EN |                 |
| Grove   |               |               | PY_GROVE_OUT_EN |

- DCDC3V3_EN_PP(PY_MPWR_EN): 3V3_L2 层电源开关
- LDO3V3_EN_PP(RTCPY_RGB_PWR_EN_IRQ): RGB LED 供电开关
- BOOST5V_EN_PP(PY_GROVE_OUT_EN): Grove 拓展接口供电方向控制

| M5PM1   | PYG0      | PYG2    | PYG4         | PYG3         | PYG1     |
| ------- | --------- | ------- | ------------ | ------------ | -------- |
| E-Paper | PY_EPD_EN |         |              |              |          |
| RTC     |           | RTC_IRQ |              |              |          |
| microSD |           |         | PY_SD_DET_EN | PY_SD_PWR_EN | CARD_DEC |

- PYG0(PY_EPD_EN): 开启墨水屏供电
- PYG2(RTC_IRQ): RTC 中断信号
- PYG3(PY_SD_PWR_EN): microSD 模块供电
- PYG4(PY_SD_DET_EN): microSD 检测功能启用，使能上拉
- PYG1(CARD_DEC): microSD 插入检测
