## 描述

**Tough** 是一款工业级可编程嵌入式控制器。采用乐鑫 ESP32 主控芯片，集成 Wi-Fi 方案，搭载双核低功耗 **Xtensa® 32-bit LX6** 微处理器，主频高达 **240MHz** 。板载 **8MB PSRAM** + **16MB Flash** 内存组合，集成 2.0 英寸 320\*240 全彩 IPS 显示屏 + 高灵敏的电容触控面板 + 丰富的外设和拓展接口。全覆盖外壳具备防尘防水能力，即便处于复杂的工业应用场景中，也能够确保电路运行的稳定性。该控制器适用于工业现场控制、智能楼宇、户外数据采集节点等多种应用场景。

### 产品特性

- 基于 ESP32 开发，支持 Wi-Fi
- 16MB Flash + 8MB PSRAM
- 板载 microSD 卡槽
- 内置扬声器，I2S 功放
- IPS 显示屏，电容式触控
- AXP192 电源管理芯片
- BM8563-RTC 时钟，支持 RTC 定时唤醒
- 集成 DC 6 ~ 24V 转 5V DC-DC 降压电路
- 防尘防水设计，防紫外线塑料
- 支持 FreeRTOS
- 开发平台
  - UiFlow1
  - UiFlow2
  - Arduino IDE
  - ESP-IDF
  - PlatformIO

## 管脚映射

### LCD & microSD

| ESP32-D0WDQ6-V3 | G38  | G23  | G18 | G5  | G15 |

| AXP192   | AXP_IO4 | AXP_DC3 | AXP_LDO2 |
| -------- | ------- | ------- | -------- |
| ILI9342C | RST     | BL      | PWR      |

| ESP32-D0WDQ6-V3 | G38  | G23  | G18 | G4  |
| --------------- | ---- | ---- | --- | --- |
| microSD         | MISO | MOSI | SCK | CS  |

| ESP32-D0WDQ6-V3 | G21 | G22 | G39 |

| -------- | ------- |
| CHSC6540 | RST     |

| AXP192 | AXP_IO2 |
| ------ | ------- |
| NS4168 | SPK_EN  |

| --------------- | --- | --- |
| BM8563 (0x51)   | SDA | SCL |

| AXP192 | AXP_PWR |
| ------ | ------- |
| BM8563 | INT     |

| M5-Bus | G27 | G19 | (DC-DC 6 ~ 24V->5V) | GND |

### HY2.0_RESET

| M5-Bus | EN  | G25 | VCC | GND |
