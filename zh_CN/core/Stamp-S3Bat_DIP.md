## 描述

Stamp-S3Bat DIP 是一款集成电源管理的嵌入式核心模组。 采用 ESP32-S3-PICO-1-N8R8 主控核心，支持 2.4GHz Wi-Fi，内置 8MB Flash 与 8MB PSRAM。模组共引出 11 个 GPIO 和 1 个唤醒 IO，并配备 24P BTB 连接器（支持 DVP 接口）。板载电池供电接口用于外接 3.7V 锂电池供电。内置 M5MP1 多级电源控制设计，支持电源输入输出控制、锂电池充电控制，以及输入电压、电池电压等状态检测。同时支持外部低功耗唤醒功能，主控进入低功耗休眠状态后，可通过外部唤醒 IO 触发电源唤醒。板载可编程 RGB LED， Stamp-S3Bat DIP 版本默认焊接 2.54mm 标准间距排针，可通过插针连接 PCB 板或面包板，杜邦线连接等方式集成到应用中，帮助开发者快速构建 IoT 应用。

## 产品特性

- 集成 ESP32-S3-PICO-1-N8R8 主控
- 8MB Flash 和 8MB PSRAM
- 24P BTB 连接器（支持 DVP 接口）
- M5PM1 低功耗电源管理
- 11 x GPIO 引出 + 1x 唤醒 IO
- 1x RGB LED
- 供电方式:
  - 3.7V 电池供电
  - USB Type-C / DC 5V
- 2.54mm 标准间距排针

## 包装内容

- 1 x Stamp-S3Bat DIP
- 1 x SH1.0 电池连接线

## 操作说明

### 下载模式

将设备通过 USB 线连接至电脑，长按 `PWR` 按钮大约 3s，进入下载模式，等待烧录。

<img src="https://m5stack-doc.oss-cn-shenzhen.aliyuncs.com/1222/bat-operate_01.gif" width="30%">

### 开关机操作

- 单击 `PWR` 按钮进行开机 / 复位：

<img src="https://m5stack-doc.oss-cn-shenzhen.aliyuncs.com/1222/bat-RESET_01.gif" width="30%">

- 双击 `PWR` 按钮关机：

<img src="https://m5stack-doc.oss-cn-shenzhen.aliyuncs.com/1222/bat-poff_01.gif" width="30%">

### 外壳拆卸

!> 注意事项 | 模组在焊接或贴片前，需先拆卸外壳与 Wi‑Fi 天线，否则可能造成天线或外壳损坏，具体可参考下方拆卸教程视频进行操作。

<video class="video-container" controls><source src="https://m5stack-doc.oss-cn-shenzhen.aliyuncs.com/1222/Stamp-S3Bat-Housing-Disassembly-Tutorial.mp4" type="video/mp4"></video>

### 供电方式

模组支持以下三种供电方式，注意，若通过 3V3 接口供电，模组 EXT_5V 将无法正常输出。

- 3.7V 锂电池供电
- USB Type-C 供电
- DC 5V 供电

<img src="https://m5stack-doc.oss-cn-shenzhen.aliyuncs.com/1222/S015-power-mode.jpg" width="70%">

| M5PM1      | G0       | G1       | G2             | G3             | G4         | PY_LED_PP | CHG_EN_PP |
| ---------- | -------- | -------- | -------------- | -------------- | ---------- | --------- | --------- |
| RGB LED    | NeoPixel |          |                |                |            | PY_LED_EN |           |
| Charge     |          |          | PY_G2_CHG_STAT | PY_G3_CHG_PROG |            |           | CHG_EN    |
| 5VOUT      |          | 5VOUT_EN |                |                |            |           |           |
| Ext Wakeup |          |          |                |                | PY_G4_WAKE |           |           |

- 通过 `PY_G3_CHG_PROG` 切换充电 IC 充电电流 (低电平: 650mA / 浮空: 200mA)
