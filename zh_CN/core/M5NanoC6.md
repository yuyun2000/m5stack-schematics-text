## 描述

**NanoC6** 是 M5Stack 开发套件系列中一款微型、**低功耗**的 IoT 开发板。搭载 ESP32-C6FH4 作为 SoC ，支持 Wi-Fi 6 和 Zigbee 等无线通讯方式，并且内置红外发射器，能够便捷地控制红外物联网设备。其板载陶瓷天线可确保稳定的无线通信连接。

此外，该设备还配备可编程 RGB 灯珠，能为项目增添个性化的视觉效果。Grove 接⼝的加⼊使得 NanoC6 能够灵活扩展多种 M5 设备 （可自定义接入不同类型设备，例如遵循 UART/I2C 等通讯协议的设备），为开发者提供了丰富的硬件拓展可能性。

这款开发板适用于智能家居、工业自动化、健康监测以及物联网设备等领域，为创新项目提供了全方位的开发解决方案。

\#>Wi-Fi 6 |M5NanoC6 是首款支持 Wi-Fi 6 的核心控制板，支持 2.4 GHz Wi-Fi 6 协议 (802.11ax)，并向下兼容 802.11 b/g/n。<br/>Wi-Fi 6 协议相较之前的 Wi-Fi 协议有更高的速度、更大的容量、更低的延迟、和更强的安全性。

## 管脚映射

### IR & RGB & Button & LED

| ESP32-C6FH4 | G3  | G20    | G19        | G9     | G7        |
| ----------- | --- | ------ | ---------- | ------ | --------- |
| IR          | IR  |        |            |        |           |
| WS2812      |     | WS2812 | EN(RGBPWR) |        |           |
| BUTTON      |     |        |            | BUTTON |           |
| LED(Blue)   |     |        |            |        | LED(Blue) |

?>RGB 灯使用注意：| 使用 RGB 功能时，需将 G19 引脚设置为高电平以开启供电。

```bash
board = esp32-c6-devkitc-1
framework = arduino
upload_speed = 1500000
monitor_speed = 115200
build_flags =
    -D ARDUINO_USB_MODE=1
    -DARDUINO_USB_CDC_ON_BOOT=1
    -DCONFIG_ARDUHAL_LOG_COLORS=1
    -DCORE_DEBUG_LEVEL=5

    M5Unified=https://github.com/m5stack/M5Unified
```
