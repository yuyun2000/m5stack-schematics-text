## 描述

**Magnetic Type-C Cable**是一款带磁吸插头的 Type C 数据线，不仅使设备连接更为便捷，而且大大减少了接口的磨损，延长了使用寿命。磁吸头在遇到意外拉扯时能自动脱落，有效保护了设备免受损害。此外，M5 的设计支持多设备兼容性，能够适配各种有 Type-C 接口的设备。该数据线采用环保材料制造，符合 RoHS 环保标准。
