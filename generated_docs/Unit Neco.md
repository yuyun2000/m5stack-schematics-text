## 原理图分析

### 1. 接口与连接器 (Interface & Connectors)
电路板包含两个 HY2.0-4P Grove 接口，通常标记为 PORT A 或 PORT B，二者在物理与电气上采用并联或信号扩展连接，用于接入 M5 主控设备或其他扩展模块。

*   **引脚定义与网络分配**：

    *   **Pin 2 (5V)**：连接至 5V 电源轨 (VCC_5V)，作为板载 LED 阵列的主要供电输入。
    *   **Pin 3 (DATA)**：作为 LED 控制信号输入端。该引脚信号经由限流电阻连接至首颗 LED 的数据输入引脚 (DIN)。
    *   **Pin 4 (BTN)**：连接至板载轻触按键的一端，用于输出用户交互信号。

### 2. 核心发光电路 (Core LED Circuitry)
核心电路主要由 35 颗 WS2812C-2020 规格的 RGB LED 灯珠组成，采用单线串行级联方式驱动。

*   **级联拓扑结构**：
    *   **首颗 LED (LED1)**：其数据输入端 (DIN) 接收来自 Grove 接口 Pin 3 的控制信号。
    *   **中间链路**：前一级 LED 的数据输出端 (DOUT) 直接连接至下一级 LED 的数据输入端 (DIN)（例如：LED1_DOUT -> LED2_DIN，LED2_DOUT -> LED3_DIN），依次连接直至最后一颗 LED。
    *   **末端 LED (LED35)**：作为链路末端，其 DOUT 引脚悬空或连接至测试点（视具体布线而定），不再向下级传输。
*   **电源连接**：
    *   所有 35 颗 LED 的 VDD 引脚并联连接至 5V 电源轨。
    *   所有 35 颗 LED 的 GND 引脚并联连接至公共地。

### 3. 控制与逻辑电路 (Control & Logic Circuitry)
电路包含简单的输入逻辑电路，用于按键检测。

*   **轻触按键 (Tactile Button)**：
    *   按键的一端连接至 Grove 接口的 Pin 4 (BTN 信号线)。
    *   按键的另一端直接连接至 GND。
    *   **逻辑状态**：当按键未按下时，Pin 4 处于浮空状态（通常依赖主控端 MCU 内部的上拉电阻维持高电平）；当按键按下时，电路闭合，Pin 4 被拉低至 GND (Logic Low)，形成下降沿触发信号。

### 4. 电源管理与去耦 (Power & Decoupling)
电源电路设计主要关注 WS2812 阵列的大电流供电稳定性及高频噪声抑制。

*   **去耦电容**：
    *   在 5V 电源轨上，每一颗 WS2812C-2020 灯珠附近均配置有一颗去耦电容（通常为 100nF / 0.1uF 陶瓷电容）。
    *   这些电容并联在 VDD 与 GND 之间，用于滤除由 LED PWM 调光引起的高频电源噪声，并提供瞬态电流补充，防止电压跌落导致色彩显示异常。

### 5. 信号完整性与保护 (Signal Integrity & Protection)
为了保证通信的稳定性，信号路径中包含无源元件进行调理。

*   **数据线阻抗匹配**：
    *   在 Grove 接口 Pin 3 与首颗 LED (LED1) 的 DIN 引脚之间，串联有一颗小阻值电阻（通常为 33Ω 至 470Ω 之间）。
    *   **作用**：该电阻用于阻抗匹配，抑制信号反射（振铃效应），并限制输入电流，保护首颗 LED 的输入引脚免受瞬间尖峰电压损坏。

### 6. 模块间连接关系 (Connectivity Summary)
*   **电源域**：外部输入的 5V 电源直接驱动所有 LED，未经过降压或升压转换。
*   **信号流**：
    *   **下行 (Downstream)**：M5Host (GPIO) -> Grove Pin 3 -> 限流电阻 -> LED1 -> ... -> LED35。
