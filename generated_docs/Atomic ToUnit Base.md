### 2. IO 多路复用与开关控制 (SW-DIP8)

    *   **IO2 映射组（White Pin）**：拨码开关的偶数位（或另一组开关，如 SW2, SW4, SW6, SW8）负责将选定的 Atom GPIO 连接至 Unit 接口的 **Pin 3 (White)**。这允许用户选择另一个 GPIO 作为 Unit 的副信号线（如 SDA 或 RX）。
*   **物理断开与选择**：通过将开关拨至 OFF 位置，可以物理断开未使用的 GPIO 与 Unit 接口的连接，从而避免信号冲突（Contention）或干扰。在使用时，用户通常应确保针对 IO1 和 IO2 线路各只接通一个对应的开关。
