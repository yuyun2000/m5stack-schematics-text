*   **时钟系统**：电路设计主要依赖 MCU 内部的高速 RC 振荡器（HSI）提供系统时钟，以减少外部元件数量并适应紧凑的 Unit 封装。
*   **复位电路**：NRST 引脚配置有复位网络（通常包含上拉电阻和对地电容），确保上电时 MCU 能可靠复位。

### 2. 霍尔电磁摇杆输入电路 (Hall Effect Joystick Input)
*   **工作原理**：摇杆组件内部集成了磁铁与霍尔传感器，将物理位移转换为模拟电压信号。
*   **X 轴采样**：水平方向的位移信号输出连接至 STM32 的 **PA1 (LEFT-SW-X)** 引脚。该引脚被配置为模拟输入模式，信号直接进入 MCU 内部的 12-bit ADC（模数转换器）进行采样，将 0-3.3V 的模拟电压转换为数字量，以表示 X 轴坐标。
*   **Y 轴采样**：垂直方向的位移信号输出连接至 STM32 的 **PA2 (LEFT-SW-Y)** 引脚，同样通过 ADC 通道进行电压采样，以获取 Y 轴坐标。
*   **Z 轴按键**：摇杆的垂直按压功能通过机械微动开关实现。开关一端接地，另一端信号线连接至 **PA3 (LEFT-SW-B)**。该引脚配置为数字输入模式，并启用内部上拉电阻（或外部上拉）。常态下为高电平，按下摇杆时信号线短路至地，电平变低，触发按键事件。

    *   **SCL (White)**：I2C 时钟线连接至 STM32 的对应 I2C 时钟引脚，同样配置有上拉电阻。
*   **通讯协议**：STM32 作为 I2C 从机设备运行，固件中预设的从机地址为 **0x63**。主控设备通过该接口读取摇杆的 X/Y/Z 轴寄存器数据，或写入指令控制 RGB LED。
