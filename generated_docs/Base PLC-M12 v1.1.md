该电路板设计的核心在于支持工业环境下的宽电压输入并为 M5Core 主机及扩展模块提供稳定电源。

*   **输入级**：电路支持 **DC 9V ~ 24V** 的宽范围电压输入。电源通过 M12 工业接口或 HT3.96 端子进入 PCB。
*   **DC-DC 降压电路**：输入的高压直流电经过板载的 **DC-DC Buck Converter（降压转换器）** 芯片。该电路包含输入滤波电容、功率电感及反馈电阻网络，将 9-24V 的输入电压转换为标准的 **5V** 系统电压。
*   **功率分配网络**：

*   **收发器芯片**：电路采用标准的 **RS485 收发器芯片**（如 SP3485 或兼容型号），工作电压通常为 3.3V（由主机通过 M5-Bus 提供或本地稳压）。
*   **逻辑控制与信号路径**：
    *   **TXD 路径**：M5-Bus 的 **GPIO17** 连接至收发器的 DI (Driver Input) 引脚，负责发送数据。
    *   **RXD 路径**：收发器的 RO (Receiver Output) 引脚连接至 M5-Bus 的 **GPIO16**，负责将总线数据传输给主机。
    *   **收发控制**：收发器的 DE (Driver Enable) 和 RE (Receiver Enable) 引脚通常通过硬件逻辑控制，或默认配置为接收/发送状态，以配合 UART2 的时序。
*   **总线侧与保护电路**：
    *   收发器的 A、B 差分输出端连接至外部接口。
    *   **防护措施**：在 A、B 线上并联了 **TVS（瞬态电压抑制二极管）** 或类似保护元件，并通过匹配电阻连接，用于吸收静电（ESD）和浪涌电压，保护后端收发器芯片不受工业现场干扰信号损坏。

### 3. 接口连接与物理引脚定义
电路通过三种主要的物理接口实现对外连接与内部互联。

*   **M5-Bus (2x15 Pin 排母)**：

    *   包含电源引脚（5V, 3V3, GND, BAT, HPWR）、复位信号（RST）以及 GPIO 信号线。
    *   关键的 GPIO16/17 被专用作 RS485 通信，其余 GPIO 引脚延伸至扩展区域。
*   **HT3.96 端子 (4 Pin)**：
    *   提供板载的接线柱连接，定义通常为：**VCC (9-24V In)**, **GND**, **RS485-A**, **RS485-B**。
*   **M12 工业接口 (6 Pin)**：
    *   这是一个加固型的圆形防水接口，其内部引脚与 HT3.96 端子电气并联。
    *   承载电源输入和 RS485 通信信号，允许通过单根 M12 电缆同时实现供电与数据传输。
