## 原理图分析

*   **显示接口信号：**
    *   **Com & Seg 驱动：** SSD1309 的公用端（Common）和段（Segment）引脚通过 FPC 连接器直接驱动 OLED 像素阵列。
    *   **参考电流（IREF）：** 芯片的 IREF 引脚连接有一个特定阻值的电阻接地，用于设定 OLED 像素段的驱动电流参考值，从而控制屏幕亮度。
    *   **电压维持：** VCOMH 引脚外接电容到地，以稳定 COM 信号的高电平电压。
*   **驱动模式配置：** 芯片的 BS0、BS1、BS2 引脚通过硬件电平配置（通常为 BS1接高电平，BS0/BS2接低电平），将 SSD1309 设定为 I2C 通信模式。

### 2. 通讯接口与总线连接
*   **物理接口：** 包含两个并联的 **HY2.0-4P (Grove)** 接口，定义为 PORT.A。两个接口的引脚一一对应互连，支持 I2C 设备的菊花链（Chain）式连接。
*   **引脚定义：**
    *   Pin 1: GND（系统地）。
    *   Pin 2: 5V（电源输入）。
    *   Pin 3: SDA（I2C 数据线）。
    *   Pin 4: SCL（I2C 时钟线）。
*   **总线连接：**
    *   SCL 信号连接至 SSD1309 的 D0 引脚。
    *   SDA 信号连接至 SSD1309 的 D1 (SDAin) 和 D2 (SDAout) 引脚，实现数据的双向传输。
    *   I2C 总线上配置有上拉电阻（通常为 4.7kΩ 或 10kΩ），分别连接 SDA/SCL 与 3.3V 电源，以确保总线空闲时的高电平状态。

### 3. 逻辑控制与地址配置
*   **I2C 地址选择（SA0）：**
    *   SSD1309 的 `SA0` 引脚（在 I2C 模式下复用为从机地址位选择脚，即 D/C# 引脚）连接至一个由电阻/焊盘构成的选择电路。
    *   **默认配置：** 默认通过电阻下拉至 GND，I2C 地址配置为 `0x3C`。
    *   **可选配置：** 电路板预留了焊盘或跳线电阻位置，若将 SA0 连接至 VDD（3.3V），则 I2C 地址切换为 `0x3D`。
*   **复位电路（RES#）：** SSD1309 的 `RES#` 引脚连接有一组 RC 延时电路（电阻上拉至 3.3V，电容下地），用于在系统上电瞬间产生低电平脉冲，完成驱动芯片的硬件复位。

### 4. 电源管理系统
*   **电源输入与稳压：**
    *   外部 5V 电源通过 Grove 接口输入。
    *   **LDO 稳压：** 使用低压差线性稳压器（LDO）将 5V 输入电压转换为 **3.3V**，为 SSD1309 的逻辑部分（VDD/VDDIO）及总线上拉电阻供电。输入和输出端均并联有陶瓷电容进行滤波。
*   **OLED 面板驱动电压（VCC）：**
    *   SSD1309 内部集成了电荷泵电路。电路图中 SSD1309 的相关引脚（如 C1P/C1N, C2P/C2N）之间连接有飞跨电容（Flying Capacitors）。
    *   通过内部电荷泵将 3.3V 升压至 OLED 面板所需的高压（VCC，通常约 12V-13V），无需外部独立的高压升压芯片。

### 5. 其他电气特性
*   **逻辑电平：** 系统内部逻辑电平统一为 3.3V。由于 I2C 总线上拉至 3.3V，该模块可直接与 3.3V 主控系统兼容；若连接 5V 逻辑主控，依赖 I2C 的开漏特性及上拉电阻工作。
*   **接地：** 所有模块共用 GND 网络，OLED 面板的阴极通过 FPC 接口最终汇入系统地。
