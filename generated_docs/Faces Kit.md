## 原理图分析

**主控系统与架构**
核心处理单元采用 ESP32-D0WDQ6 SoC，具备双核 Xtensa® 32-bit LX6 微处理器，运行频率高达 240MHz，搭配 16MB SPI Flash 存储器。该主控负责系统的所有逻辑处理、无线通信（Wi-Fi/Bluetooth）以及外设调度。

**电源管理与供电系统**
电源管理电路核心为 IP5306 多功能电源管理 IC。该芯片集成了锂电池充电管理与同步升压转换器，负责管理 3.7V/600mAh 聚合物锂电池的充放电过程，并将电池电压升压至 5V 供系统使用。IP5306 通过 I2C 总线与 ESP32 通信，允许主控读取电量状态并配置电源寄存器。系统支持通过 USB Type-C 接口进行供电和充电。

**人机交互系统 (显示与面板)**
1.  **显示驱动**：视觉输出采用 2.0 英寸 IPS 彩色液晶屏，驱动芯片为 ILI9342C。屏幕通过 SPI 总线与 ESP32 连接，引脚分配如下：MOSI (G23), CLK (G18), CS (G14), DC (G27), RST (G33)。背光控制 (BL) 连接至 G32，支持 PWM 调光。
2.  **Faces 面板控制器**：可更换的功能面板（GameBoy 键盘、计算器键盘、QWERTY 全键盘）内部集成 ATmega328 (MEGA328) 协处理器。该协处理器负责扫描按键矩阵，并作为 I2C 从机设备（地址 0x08）运行。ESP32 作为 I2C 主机，通过 GPIO21 (SDA) 和 GPIO22 (SCL) 读取面板输入的键值数据，实现输入逻辑与主控的分离。

**传感器与音频模块**
1.  **惯性导航与地磁**：板载集成 MPU6886 6 轴 IMU（加速度计 + 陀螺仪）与 BMM150 3 轴地磁传感器。两者均挂载于系统 I2C 总线（SDA: G21, SCL: G22），提供姿态解算与方位检测功能。
2.  **音频输出**：音频电路采用 1W-0928 扬声器，直接连接至 ESP32 的 DAC 输出引脚 G25。音频信号通过 DAC 数模转换后驱动扬声器发声。

**外部存储接口**
系统板载 MicroSD (TF) 卡插槽，用于大容量数据存储。存储接口通过标准 SPI 总线通信，信号线路包括：MOSI (G23), MISO (G19), CLK (G18)。片选信号 (CS) 独立分配至 G4，与显示屏共享 SPI 总线但通过 CS 信号分时复用。

**通讯接口与总线扩展**
1.  **USB 调试通讯**：采用 USB Type-C 物理接口，后端连接 USB 转 TTL 串口芯片（CP2104 或 CH9102），实现程序下载与串口调试。
2.  **GROVE 扩展接口**：提供标准的 HY2.0-4P 接口，分为三类功能定义：
    *   Port A (I2C): 连接 G21/G22。
    *   Port B (I/O): 连接 G26/G36 (DAC/ADC)。
    *   Port C (UART): 连接 G16/G17 (UART2)。
3.  **M5-Bus 总线**：通过扩展母座引出 ESP32 的主要 GPIO 资源，包括：
    *   UART0: G1 (TXD0), G3 (RXD0)。
    *   UART2: G16 (TXD2), G17 (RXD2)。
    *   I2C 总线: G21 (SDA), G22 (SCL)。
    *   SPI 总线: G18 (CLK), G19 (MISO), G23 (MOSI)。
    *   其他 GPIO 及电源引脚（5V, 3.3V, BAT, GND）。

**信号路径与连接关系**
系统内部主要依靠 I2C 和 SPI 两条总线进行互联。I2C 总线（G21/G22）连接了电源管理 IC (IP5306)、IMU (MPU6886)、地磁计 (BMM150) 以及外部 Faces 面板的协处理器 (ATmega328)。SPI 总线（G18/G19/G23）则主要负责高速数据传输，连接 LCD 屏幕（输出）和 MicroSD 卡（读写），通过各自独立的片选信号（G14 用于 LCD，G4 用于 SD 卡）避免总线冲突。
