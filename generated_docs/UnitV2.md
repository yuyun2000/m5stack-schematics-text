*   **USB 端口独占**：Type-C 接口的 USB 2.0 差分信号线（D+/D-）**专用于连接 SR9900**，并未直接路由至 SSD202D 的 USB OTG 控制器。这意味着 SSD202D 无法直接作为 USB Gadget 设备被电脑识别，必须依赖 SR9900 建立的网络连接进行 SSH 通信或数据传输。
