*   **Pin 1 (GND/Black):** 连接至系统公共地（Ground），作为电源回路和信号参考地。

### 2. 核心驱动电路设计
该单元采用**低侧开关（Low-Side Switch）**拓扑结构来驱动大功率感性负载（N20 电机）。
*   **开关元件:** 使用一颗 N-Channel MOSFET（N 沟道场效应管，常见型号如 AO3400A 或类似 SOT-23 封装逻辑电平管）。
*   **连接方式:**
    *   MOSFET 的**源极（Source）**直接连接至 GND。
    *   MOSFET 的**漏极（Drain）**连接至 N20 电机的负极。
    *   MOSFET 的**栅极（Gate）**通过限流电阻（通常为小阻值或直连）连接至 DIN 接口。
*   **控制逻辑:**
    *   当 DIN 输入**高电平**时，Vgs > Vth，MOSFET 导通，电机回路形成通路（5V -> 电机 -> Drain -> Source -> GND），电机旋转产生震动。
    *   当 DIN 输入**低电平**时，MOSFET 截止，电机断电停止。
    *   通过向 DIN 输入 **PWM（脉宽调制）** 信号，通过调节占空比（Duty Cycle）快速切换 MOSFET 的导通与截止状态，从而控制电机的平均有效电压，实现震动强度的无级调节。

### 3. 电源与保护电路
针对 N20 电机作为感性负载的特性，电路中包含关键的保护设计：
*   **续流二极管（Flyback Diode）:** 在电机正负极两端并联了一颗二极管（通常为 1N5819 等肖特基二极管或 1N4148）。
    *   **连接极性:** 二极管阴极（Cathode）接 5V，阳极（Anode）接 MOSFET 漏极（电机负极）。
    *   **功能:** 当 MOSFET 从导通变为截止时，电机线圈因自感现象会产生反向感应电动势（高压尖峰）。续流二极管为这部分能量提供泄放回路（回流至电源端），将反向电压钳位在安全范围内，防止高压击穿 MOSFET。
*   **电源滤波:** 在 5V 输入端对地并联有去耦电容（通常为 100nF 至 10uF 陶瓷电容）。该电容用于吸收电机换向时产生的电源噪声，并补充电机启动瞬间的大电流需求，防止电源电压跌落影响主控稳定性。

### 4. 信号稳定性与偏置设计
为了确保电路在各种工况下的确定性与稳定性：
*   **栅极下拉电阻:** 在 DIN 信号线（MOSFET 栅极）与 GND 之间通常并联一颗下拉电阻（阻值通常为 10kΩ 或 100kΩ）。
    *   **作用:** 确保在主控 GPIO 处于高阻态、复位或未连接时，MOSFET 栅极保持低电平，强制 MOSFET 处于关闭状态，防止电机因干扰信号意外误动作。

### 5. 硬件功能信号链条
完整的硬件信号控制流程如下：
1.  **信号输入:** 主控 MCU 产生 PWM 或数字逻辑信号，经由 Grove 线缆传输至 DIN 引脚。
2.  **开关控制:** 信号电压作用于 MOSFET 栅极，控制器件的导通阻抗。
3.  **电流回路:** MOSFET 导通，5V 电源电流流经电机线圈，驱动 N20 电机转子旋转。
4.  **机械转换:** 电机轴带动金属偏心轮（Eccentric Rotating Mass, ERM）高速旋转（规格约 8800 RPM）。
5.  **震动输出:** 偏心轮产生的离心力导致整个 Unit 模块发生物理位移震动，实现触觉反馈。
