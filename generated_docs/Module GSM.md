### 2. 电源管理与供电电路
*   **主供电路径**：模块从 M5-Bus 的 **5V** 引脚获取主电源。

*   **UART 串行通讯**：
    *   **TXD 路径**：主控（M5Core）的 **GPIO 17** 连接至 M6315 的 RXD 引脚。

*   **电平匹配**：由于 M6315 的 I/O 电平（通常为 2.8V 或 VBAT 域）与 ESP32 主控的 3.3V 电平不完全一致，信号路径上通常串联电阻或使用电平转换晶体管/芯片，以确保信号完整性并保护模组 I/O 口。
