    *   在 M5-Bus 的物理引脚定义中，HPWR 通常对应 **Pin 25, Pin 27, Pin 29**。这种设计允许堆叠在总线上的其他大功率模块（如电机驱动、继电器模块）直接从底座获取高压电源，而无需通过主控的 5V 稳压器，实现了逻辑电源与动力电源的分流。
