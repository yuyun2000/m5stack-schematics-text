*   **摄像头配置总线 (SCCB)**：专用于 ESP32 与 OV3660 通信 (G23/SIOC, G25/SIOD)，确保图像传感器寄存器配置的高速与独立。

### 2. 电源路径管理与保护特性的补充
*   **VBUS 强制唤醒机制**：电源控制逻辑包含 VBUS 检测电路。当 USB 接口接入 5V (VBUS) 时，硬件电路会强制拉高电源使能信号（或旁路 MOSFET 开关），使得系统忽略 BAT_HOLD 的状态而强制保持开机。这确保了在充电或固件烧录场景下设备不会意外关机。
*   **瞬态电压跌落与 BOD 处理**：由于 OV3660 模组启动与 Wi-Fi 射频校准（RF Calibration）同时发生时会产生高瞬态电流，可能导致 3.3V 电源轨产生瞬间压降。虽然电路设计包含滤波电容，但该压降仍可能触发 ESP32 默认的高灵敏度掉电检测（Brownout Detect）。因此，硬件系统的稳定性强依赖于软件配置禁用硬件 BOD 功能（`RTC_CNTL_BROWN_OUT_REG`），以防止在大负载启动时的误复位。
