## 原理图分析

以下是 M5Stack Core2 For AWS (SKU: K010-AWS) 的硬件原理图详细技术描述：

### 1. 核心控制系统 (Core Control System)
*   **主处理器**：电路核心采用 **ESP32-D0WDQ6-V3** SoC，基于双核 Xtensa® 32-bit LX6 架构，主频高达 240MHz。
*   **存储单元**：集成 **16MB SPI Flash** 用于程序存储，以及 **8MB PSRAM** 用于扩展运行内存，满足 AWS IoT 应用对内存的需求。
*   **调试与下载接口**：通过 USB Type-C 接口连接，采用 **CP2104** 或 **CH9102** USB 转串口芯片实现与 PC 的通信。转换后的 UART 信号连接至 ESP32 的 **UART0** 接口，具体引脚定义为 RXD 连接 GPIO3，TXD 连接 GPIO1。电路包含自动下载电路（利用 DTR/RTS 信号控制 ESP32 的 EN 和 IO0）。

### 2. 电源管理系统 (Power Management System)
*   **PMU (电源管理单元)**：核心电源管理采用 **AXP192** 芯片。ESP32 通过 I2C 总线 (SDA: GPIO21, SCL: GPIO22) 对其进行配置和控制。
*   **电源分配与控制**：
    *   **DC-DC1**：供给 MCU 主电源 (3.3V)。
    *   **AXP_LDO2**：供给 LCD 屏幕逻辑电源。
    *   **AXP_LDO3**：驱动 **1027 震动马达**，实现触觉反馈控制。
    *   **AXP_DC3**：控制 LCD 背光 (BL) 的电压，实现亮度调节。
    *   **AXP_IO1**：控制机身绿色 LED 电源指示灯。
    *   **AXP_IO2**：连接至音频功放 (NS4168) 的使能端 (EN)，用于控制功放开关。
    *   **AXP_IO4**：作为复位信号源，同时连接至 LCD 屏幕和触摸屏芯片的 Reset 引脚。
*   **供电与充电**：
    *   **内置电池**：板载 500mAh 锂聚合物电池。
    *   **底座充电**：通过底部的 Pogo Pin 接口供电，底座电路集成 **TP4057** 线性充电芯片，实现磁吸充电功能。
    *   **升压电路**：采用 **SY7088** DC-DC 升压芯片，负责将电池电压升压至 5V，供给内部 5V 总线及外部扩展接口。

### 3. 安全加密模块 (Secure Element) - AWS 专用
*   **硬件加密**：板载 Microchip **ATECC608B-TNGTLSU-G** (Trust&GO) 安全芯片。
*   **连接方式**：挂载于系统 I2C 总线，I2C 地址为 **0x35**。该芯片用于存储设备证书、私钥并执行硬件级加密运算，支持 AWS IoT Core 的安全连接与认证。

### 4. 显示与人机交互 (Display & HMI)
*   **显示屏**：采用 2.0 英寸 IPS 彩色 LCD，驱动芯片为 **ILI9342C**。
    *   **接口协议**：SPI 接口。
    *   **信号连接**：MOSI (GPIO23), SCK (GPIO18), MISO (GPIO38), CS (GPIO5), DC (GPIO15)。
*   **触摸屏**：集成电容式触摸面板，控制器为 **FT6336U**。
    *   **接口协议**：I2C 接口，地址 **0x38**。
    *   **信号连接**：中断信号 (INT) 连接至 GPIO39。复位信号与 LCD 共用 AXP_IO4。
*   **虚拟按键**：屏幕正面玻璃盖板下的三个圆形触摸热区映射为虚拟按键。
*   **物理按键**：侧边设有电源键（连接至 AXP192）和复位键（RST）。

### 5. 音频系统 (Audio System)
*   **音频输出**：使用 **NS4168** 单通道 I2S D 类音频功率放大器。
    *   **接口协议**：I2S 接口。
    *   **信号连接**：BCLK (GPIO12), LRCK (GPIO0), DATA (GPIO2)。
    *   **控制**：使能端由 AXP_IO2 控制。
*   **音频输入**：集成 **SPM1423** PDM 数字麦克风。
    *   **接口协议**：PDM 接口。
    *   **信号连接**：CLK (GPIO0), DATA (GPIO34)。注意：麦克风时钟线与功放的 LRCK 共用 GPIO0。

### 6. 传感器与 RTC (Sensors & RTC)
*   **惯性测量单元 (IMU)**：板载 **MPU6886** 六轴姿态传感器（3轴加速度计 + 3轴陀螺仪）。
    *   **连接方式**：I2C 总线，地址 **0x68**。
*   **实时时钟 (RTC)**：采用 **BM8563** 芯片。
    *   **连接方式**：I2C 总线，地址 **0x51**。
    *   **中断**：INT 引脚连接至 AXP192 的电源唤醒逻辑，支持定时唤醒功能。

### 7. 外设接口与扩展 (Peripherals & Expansion)
*   **TF Card 插槽**：支持 MicroSD 卡扩展。
    *   **连接方式**：SPI 总线（与 LCD 共用 SPI 信号线），片选信号 (CS) 连接 GPIO4。
*   **RGB LED 灯带**：底座板载 10 颗 **SK6812** 侧发光编程 LED。
    *   **控制信号**：通过 GPIO25 单线串行控制。
*   **PORT 扩展接口**：提供 HY2.0-4P 接口用于连接 Unit 模块。
    *   **PORT A (红色)**：I2C 扩展，信号线为 GPIO32 (SDA), GPIO33 (SCL)。
    *   **PORT B (黑色)**：通用 I/O 或模拟接口，信号线为 GPIO26 (DAC/ADC), GPIO36 (ADC)。
    *   **PORT C (蓝色)**：UART 扩展 (UART2)，信号线为 GPIO13 (RXD2), GPIO14 (TXD2)。
*   **M5-Bus 总线**：通过 2x15 Pin 排母引出系统主 SPI, I2C, UART 及部分 GPIO、5V、3.3V、BAT 电源，用于堆叠功能模块。
*   **Pogo Pin**：包含 VCC_5V, GND, HPWR (用于充电检测), SDA, SCL，支持数据通讯与充电。
