### 2. 电源管理电路 (DC-DC 降压)

*   **降压转换 (Buck Converter)**：核心电源管理采用开关型降压稳压芯片（DC-DC Buck Converter）。
    *   **拓扑结构**：电路包含输入滤波电容、开关稳压芯片、功率电感（Inductor）、续流二极管（Schottky Diode）以及输出滤波电容。

    *   **电源路径**：降压后的 5V 电源直接连接至 M5-Bus 总线的 **5V 引脚 (Pin 28)**，从而为主机（Core）及其它堆叠模块供电。同时，GND 网络与系统共地。

### 3. RS485 通讯模块电路
*   **物理接口**：RS485 的差分信号线 A 和 B 连接至 **HT3.96-4P** 端子的对应引脚，用于外部双绞线连接。

    *   **差分侧**：芯片的 A、B 引脚连接至输出端子。A/B 线之间通常预留有终端电阻位或偏置电阻设计。
    *   **TTL 侧**：芯片的 **RO (Receiver Output)** 和 **DI (Driver Input)** 引脚连接至 M5-Bus 总线上的特定 UART 通讯引脚（通常映射至 ESP32 的 UART2 或自定义 GPIO）。
    *   **方向控制**：芯片的 **RE (Receiver Enable)** 和 **DE (Driver Enable)** 引脚短接后连接至 M5-Bus 的 GPIO 引脚，用于控制数据传输的方向（接收/发送切换）。

### 4. M5-Bus 接口与管脚映射
电路通过标准的 2x15 排母（M5-Bus）与主机连接，主要电气连接如下：
*   **电源网络**：
    *   **HPWR (Pin 25, 27, 29)**：与外部电源输入或电池高压侧相连（视具体跳线或内部走线而定）。
    *   **5V (Pin 28)**：连接板载 DC-DC 转换器的 5V 输出，作为系统的供电输入点。
    *   **3V3 (Pin 12)**：由主机端降压后回传，底座本身主要使用 5V 或外部输入电压。
    *   **BAT (Pin 30)**：连接至电池正极网络，用于监测电池电压或充电路径。

*   **控制与信号**：

    *   **I2C 总线**：**SDA (Pin 17)** 和 **SCL (Pin 18)** 在总线上直通，并引出至板载的扩展焊盘，支持挂载额外的 I2C 设备。
