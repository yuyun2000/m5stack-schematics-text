*   **信号转换逻辑：**
    *   **发送路径 (TX)：** 来自 Atom 主机的 TTL 电平信号从 Atom Bus 的 **Pin 2 (UART_TX)** 进入 MAX232 的 TTL 输入端 (T1IN)，经内部驱动转换为 RS232 电平后，从 T1OUT 引脚输出至 VH-3.96 接口的 TX 端。
