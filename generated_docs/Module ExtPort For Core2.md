Module ExtPort For Core2 是一款基于 M5-Bus 总线架构的无源信号扩展与分配模块。其核心电路设计思想是通过 **物理跳线（拨码开关）逻辑** 实现总线 GPIO 的灵活复用，解决了 Core2 主机引脚资源固定与外设需求多变之间的矛盾。电路主要由 M5-Bus 总线接口、固定信号路由电路、可配置信号选择电路（拨码开关组）以及电源分配网络组成。

#### 2. M5-Bus 总线接口
模块通过标准的 30-Pin M5-Bus 堆叠接口与 Core2 主机连接。该接口作为系统的信号源与电源输入端，负责传输 5V、3.3V 电源轨以及 GPIO 信号。

- **电源接入**：总线提供的 BAT/5V 和 3.3V 直接接入模块的电源层。

- **PORT.C (UART/GPIO)**：
  - 接口类型：HY2.0-4P。
  - **Pin 3 (Yellow)**：直接连接 M5-Bus 的 **G14** (TXD)。
  - **Pin 4 (White)**：直接连接 M5-Bus 的 **G13** (RXD)。
  - 该端口专用于串行通讯。

##### 3.2 可配置信号接口（PORT.D 与 PORT.E）
这两组接口采用 **拨码开关选择网络（DIP Switch Selection Network）**，实现了“多选一”的物理信号映射。每个端口包含两个独立的信号通道（Pin 3 和 Pin 4），每个通道由一组 3 位拨码开关控制。

- **PORT.D 信号映射**：
  - **Pin 4 (White) 选择电路**：通过 3 位拨码开关分别连接至 **G34**（默认）、**G22**、**G3**。
  - **Pin 3 (Yellow) 选择电路**：通过 3 位拨码开关分别连接至 **G35**（默认）、**G21**、**G1**。
  
- **PORT.E 信号映射**：
  - **Pin 4 (White) 选择电路**：通过 3 位拨码开关分别连接至 **G27**（默认）、**G2**、**G0**。
  - **Pin 3 (Yellow) 选择电路**：通过 3 位拨码开关分别连接至 **G19**（默认）、**G25**、**G2**。

#### 4. 电源管理与分配
模块本身不包含电压调节器（LDO）或电源管理芯片（PMIC），主要通过铜箔层进行电源分配。

- **接地（GND）**：所有接口的 **Pin 2 (Black)** 及排针的 GND 引脚汇聚至模块的大面积铺地层，确保信号回流路径完整，减少干扰。

- **短路风险**：由于 3 个输入信号源（GPIO）直接并联至同一个输出端，若用户误将同一组拨码开关中的多个位同时置为 `ON`，将导致不同的处理器 GPIO 引脚在物理上直连短路。当这些 GPIO 输出不同逻辑电平（一高一低）时，会产生不可控的回路电流，极大概率损坏 Core2 主控芯片的 I/O 单元。因此，电路逻辑严格限制**单路单通**。
