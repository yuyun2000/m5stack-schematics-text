系统采用模块化堆叠设计，核心逻辑控制由 **Atom-Lite (ESP32-PICO)** 主控模块承担。DTU Base 板通过 2.54mm 排母与 Atom-Lite 连接，获取控制信号与 5V 电源。
*   **主控接口**：Atom-Lite 的 GPIO 引脚通过排母网络分配至各个功能子模块。
*   **逻辑电平**：系统主要逻辑电平为 3.3V，与 ESP32 I/O 标准一致。

### 2. LTE CAT1 通讯模组电路 (SIM-A7680C)
核心通讯功能由 SIMCom **SIM-A7680C** 模组实现，电路包含电源滤波、UART 数据链路、控制逻辑及射频前端。

*   **电源输入 (VBAT)**：模组供电端并联大容量钽电容与多层陶瓷电容（MLCC），以应对 LTE 模组在发射瞬间（Tx Burst）产生的高峰值电流，防止电压跌落，工作电压通常稳定在 3.8V-4.2V 范围内。
*   **UART 数据链路**：
    *   **RXD (模组)**：连接至 Atom 主控的 **GPIO22 (TX)**，用于接收来自主控的 AT 指令。
    *   **TXD (模组)**：连接至 Atom 主控的 **GPIO19 (RX)**，用于向主控发送数据及响应。
    *   通讯线路串联小阻值匹配电阻以抑制反射干扰。
*   **控制逻辑**：
    *   **PWRKEY**：连接至驱动三极管（或 MOSFET）的基极/栅极，由 Atom 主控 GPIO 控制，用于实现模组的软件开机与关机序列。
    *   **RESET**：连接至硬件复位电路，允许主控通过 GPIO 拉低电平强制复位模组。
*   **状态指示**：
    *   **NETLIGHT**：引脚驱动一颗状态 LED，通过闪烁频率指示网络注册状态（未连接、搜索中、已连接）。
*   **RF 射频电路**：
    *   模组 RF 引脚连接至 **SMA 天线座**。
    *   信号路径包含 $\pi$ 型（Pi-network）阻抗匹配电路（预留电感/电容焊盘），以确保 50$\Omega$ 特性阻抗匹配，优化天线驻波比（VSWR）。

### 3. SIM 卡接口电路
采用 MicroSIM 卡座，提供标准的 6 针接口连接 SIM-A7680C。
*   **信号线**：SIM_CLK, SIM_DATA, SIM_RST 直接连接模组对应引脚。
*   **电源 (SIM_VDD)**：配置去耦电容（通常为 100nF/33pF 组合）滤除高频噪声。
*   **防护设计**：数据与时钟线上配置 ESD 静电防护阵列（TVS Diode Array），防止插拔 SIM 卡时的静电击穿模组引脚。

### 4. RS485 通讯接口电路
RS485 物理层由 **SP3485EN-L/TR**（或同类 3.3V 供电收发器）实现，支持工业级总线通讯。

*   **收发控制**：
    *   **DI (Driver Input)**：连接 Atom **GPIO23**，作为发送数据端。
    *   **RO (Receiver Output)**：连接 Atom **GPIO33**，作为接收数据端。
    *   **DE/RE (Enable)**：通常连接至专用 GPIO 或自动收发切换电路，用于控制总线的数据流向（半双工模式）。
*   **总线接口**：
    *   输出端连接至 **VH3.96-4P** 工业端子。
    *   A/B 差分线之间通常并在 120$\Omega$ 终端电阻（或预留位置）。
    *   配置 TVS 二极管或气体放电管进行防雷击浪涌保护。
*   **电源输入**：VH3.96 端子包含 12V 输入引脚，直接接入板载电源管理电路。

### 5. 电源管理系统 (PMS)
系统设计了双路电源输入与多级电压转换网络，以适应 5V USB 供电与 12V 工业供电。

*   **输入源选择**：
    *   **DC 12V 输入**：来自 RS485 接口（VH3.96-4P），用于工业现场供电。
    *   **DC 5V 输入**：来自 Atom-Lite 的 USB Type-C 接口。
*   **电压转换 (DC-DC Buck)**：
    *   电路包含一颗高效率同步降压 DC-DC 转换器芯片。
    *   **12V -> 5V/VSYS**：将外部 12V 降压至 5V，为 Atom-Lite 供电（Atom 内部再通过 LDO 转 3.3V）。
    *   **供电分支**：
        1.  **3.8V-4.2V 电源轨**：通过大电流 LDO 或可调 DC-DC 专门为 SIM-A7680C 模组供电，确保 RF 发射时的电流需求。
        2.  **3.3V 电源轨**：由 Atom-Lite 回供或板载 LDO 生成，为 SP3485、I2C 接口及逻辑电平提供基准。

### 6. 外设接口与扩展
*   **I2C 总线**：

*   **GPIO 扩展**：

### 1. RS485 硬件自动收发流控电路
*   **流控机制修正**：根据《管脚映射表》，Atom 主控仅分配了 **G23 (TX)** 和 **G33 (RX)** 用于 RS485 通讯，未分配 GPIO 用于 SP3485 的 DE/RE（发送/接收使能）引脚。这表明原理图中包含**硬件自动收发切换电路**。
*   **电路原理**：该电路通常利用 TX 信号线驱动三极管或逻辑非门，动态控制 DE/RE 引脚。当 TX 发送低电平（Start Bit/Data 0）时，电路拉高 DE/RE 使能发送；当 TX 为高电平（Stop Bit/Data 1/Idle）时，电路快速切换回接收模式（高阻态），利用总线上下拉电阻维持逻辑高电平。这简化了软件开发，无需通过代码控制时序。

### 2. M5-Bus 总线物理层定义
*   **接口形式**：系统通过 M5Stack 标准的 **M-BUS** 接口（通常为 2x15 或类似排母结构）连接 Atom-Lite 与 DTU Base 底板。
*   **电源与信号分布**：
    *   **系统电源**：5V 总线贯穿连接 Atom-Lite 的 USB 输入与底板电源管理电路，实现双向供电支持。
    *   **信号分组**：
        *   **Group A (UART)**: Pin 2/3 (G22/G19) 专用于 LTE 模组通讯。
