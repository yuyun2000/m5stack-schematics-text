### 1. 整体电源拓扑与输入保护
*   **AC 输入回路**：电路通过符合 PSE 标准的日标/美标插头引入交流电（AC 100-120V）。火线（L）输入端串联有**保险丝（Fuse）**，用于过流和短路保护，额定电流匹配设备最大负载能力（通常设计为稍高于 10A 的熔断值）。
*   **浪涌保护**：在火线（L）与零线（N）之间跨接有**压敏电阻（MOV）**，用于吸收雷击或电网波动产生的高压浪涌，保护后端电路。
*   **电源转换（AC-DC）**：交流电经过一微型 **AC-DC 隔离电源模块**，将高压交流电转换为直流 **5V**。该 5V 电源主要为 ATOM 主控、继电器线圈及外设接口供电。由于 ATOM 具有外露的 USB 接口和 Grove 接口，该 AC-DC 模块通常采用隔离拓扑以确保用户安全。

### 2. 核心计量电路 (HLW8032)
该部分电路负责采集电压、电流与功率数据，并位于非隔离的高压侧（Hot Ground）。
*   **芯片供电**：HLW8032 的 VDD 引脚供电通常由 AC 输入端通过阻容降压（RC Dropper）或独立的非隔离稳压电路提供，使其地电位（GND）参考交流电的零线或火线，以便进行直接测量。
*   **电压采样**：AC 火线（L）通过高阻值的**分压电阻网络**（通常由多个贴片电阻串联以耐受高压）连接至 HLW8032 的电压采样输入引脚（VP/VN），实现电压信号的衰减与采集。
*   **电流采样**：在主回路的零线或火线路径中串联有一枚低阻值的**锰铜采样电阻（Shunt Resistor）**（例如 1mΩ 或 2mΩ）。负载电流流过该电阻产生微弱压降，该差分信号直接接入 HLW8032 的电流采样引脚（IP/IN）。
*   **数据通信与隔离**：
    *   HLW8032 的 **TX** 引脚输出 TTL 电平的 UART 异步串行数据（波特率 4800bps）。
    *   **光耦隔离**：由于 HLW8032 位于强电侧，而 ATOM 主控位于弱电侧，信号传输路径中设有一枚**光电耦合器（Optocoupler）**。HLW8032 的 TX 驱动光耦内部的发光二极管，光耦次级的集电极/发射极输出信号连接至 ATOM 主控的 **G22** 引脚。此设计实现了强弱电的电气隔离，保护主控及 USB 端口。
    *   HLW8032 被配置为 UART 模式，TX 引脚周期性发送数据包。

### 3. 继电器控制电路
*   **继电器规格**：采用 5V 线圈电压的单刀单掷（SPST）或单刀双掷（SPDT）继电器，触点容量满足 AC 120V/10A。
*   **物理断开**：继电器的常开触点（NO）串联在 AC 输出插座的火线（L）路径上。
*   **驱动逻辑**：
    *   控制信号来自 ATOM 主控的 **G23** 引脚。
    *   **驱动级**：G23 输出高电平信号经过限流电阻，驱动 **NPN 三极管**（或 N-MOSFET）导通。三极管集电极连接继电器线圈的负极，线圈正极接 5V 电源。
    *   **续流保护**：在继电器线圈两端反向并联有一枚**二极管（Flyback Diode）**，用于在三极管截止瞬间吸收线圈产生的反向电动势，防止击穿驱动管。

### 4. 接口与连接器电路
*   **ATOM 插槽（2xN Header）**：
    *   **5V & GND**：连接至 AC-DC 模块输出的隔离 5V 电源及系统地。
    *   **G22 (RX)**：连接至光耦输出端，接收来自 HLW8032 的计量数据。
    *   **G23 (Relay)**：连接至继电器驱动电路输入端。
    *   **3V3**：由 ATOM 内部 LDO 输出，用于拉高部分逻辑电平。
*   **AC 输出插座**：
    *   零线（N）直接连通输入端零线。
    *   火线（L）经过继电器触点控制后连接至插座 L 端。
    *   地线（G）通常为直通设计（若包含接地脚）。
*   **HY2.0-4P (Grove 接口)**：
    *   引脚定义通常为 GND, 5V, G33 (或其他 GPIO), Gxx。
    *   该接口电源取自隔离后的 5V，信号线直接连接 ATOM 的 GPIO，支持外部传感器或按键输入。

### 5. 信号流向总结
1.  **控制流**：ATOM (G23) -> 三极管驱动 -> 继电器动作 -> AC 输出通断。
2.  **数据流**：AC 负载 -> 锰铜电阻/分压网络 -> HLW8032 采样 -> HLW8032 TX -> 光耦隔离 -> ATOM (G22 RX)。
