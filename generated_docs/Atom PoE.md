### 2. PoE 电源模块与供电电路
该部分电路负责从以太网线缆中提取电能并转换为系统电压，符合 IEEE802.3 af 标准。

*   **电压转换**：
    *   **主电源**：PoE 模块将输入的 48V（典型值）以太网电源降压稳压为 **5V DC**。此 5V 电源通过 M5Bus 总线直接供给 ATOM 主控。
    *   **二级稳压**：系统包含一颗 LDO（低压差线性稳压器），将 5V 电源进一步转换为 **3.3V**，专门为 W5500 芯片及其外围逻辑电路供电。
*   **保护电路**：在 PoE 输入端通常配置有瞬态电压抑制二极管（TVS）或整流桥电路，用于防止极性反接和浪涌电压对后端电路造成损坏。

### 3. 网络接口 (RJ45)
RJ45 连接器集成了网络变压器（Magnetics），是连接以太网线缆与 PCB 的物理接口。
*   **信号传输**：连接器内部的变压器线圈对差分信号进行耦合，实现阻抗匹配和共模噪声抑制。变压器的次级侧直接连接 W5500 的 PHY 引脚。
*   **中心抽头**：变压器的中心抽头（Center Taps）被引出用于 PoE 取电（如果采用 Phantom Power 模式），或者通过连接器的空闲引脚（4/5, 7/8）进行取电。
*   **状态指示**：RJ45 接口集成了 Link（连接）和 Activity（数据传输）LED 指示灯，其控制引脚连接至 W5500 的状态输出引脚。

### 4. ATOM 总线接口 (M5Bus)
该接口是 Atom PoE 底座与 ATOM 主控连接的物理桥梁，采用标准的 M5Stack ATOM 接口定义。
*   **电源分配**：
    *   **5V**：PoE 模块输出的 5V 电源由此引脚进入 ATOM 主控，为其供电。
    *   **3.3V**：连接至系统 3.3V 导轨。
    *   **GND**：系统的公共地平面，连接 PoE 模块的输出地和 W5500 的数字/模拟地。
*   **SPI 总线布局**：G22, G19, G23, G33 信号线通过此接口直接路由至 ATOM 的 GPIO，确保高速信号传输的完整性。

### 5. 逻辑电平与辅助电路
*   **去耦电容**：在 W5500 的每个电源引脚（3.3V）及 PoE 模块的输出端均布置了去耦电容（如 0.1uF 和 10uF 组合），以滤除高频噪声并稳定电压。
*   **阻抗匹配**：在 W5500 的 TX/RX 差分线上配置了终端电阻（通常为 49.9Ω 或 50Ω），以匹配以太网传输线的阻抗，减少信号反射。
