*   **电源网络**：

*   **时钟与复位**：
    *   由于 PICO-D4 内部已集成晶振，外部无需主晶振电路。

*   **RF 射频电路**：芯片的 LNA_IN 端口直接输出 RF 信号，经过板载的 π 型匹配网络（电感与电容组合）连接至 3D 2.4GHz 天线，用于 Wi-Fi 和蓝牙通讯。

    *   **滤波**：输入端 (5V) 和输出端 (3.3V) 均配置了大容量陶瓷电容，确保电源轨的电压稳定性，防止负载突变引起的电压跌落。

### 4. USB 转 UART 与自动下载电路
*   **转换芯片**：板载 USB 转串口芯片（通常为 CH9102 或 CP2104 等），将 USB 的差分信号（D+/D-）转换为 TTL 电平的 UART 信号。
*   **数据通路**：
    *   芯片 TXD -> ESP32 RXD0 (GPIO3)。
    *   芯片 RXD -> ESP32 TXD0 (GPIO1)。
*   **自动下载逻辑**：
    *   利用 USB 转串口芯片的 **DTR** 和 **RTS** 信号，通过由两个三极管（或 MOS 管）构成的逻辑电路，分别控制 ESP32 的 **EN** (复位) 和 **GPIO0** (启动模式选择) 引脚。
    *   逻辑时序：确保在复位脉冲释放瞬间，GPIO0 处于低电平，从而使 ESP32 进入 UART 下载模式。
