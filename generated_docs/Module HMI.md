*   **关键信号分配**：

    *   其余 GPIO 引脚在模块内部直通，不被 STM32 占用，以保证堆叠扩展性。

### 6. 信号逻辑与通讯
*   **通讯架构**：STM32F030F4P6 作为 I2C 从机运行固件。
*   **逻辑流程**：
    1.  MCU 实时监控编码器的旋转脉冲（增量/减量）及按键状态。
    2.  内部寄存器更新当前的计数值和按键标志位。
    3.  M5Stack 主机（I2C Master）通过地址 **0x41** 发起读写请求。
