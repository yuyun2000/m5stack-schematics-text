### 2. RGB LED 灯条电路
*   **核心器件**：10 颗 SK6812（或兼容 WS2812B）侧发光 RGB LED 灯珠。
*   **连接拓扑**：采用串行级联（Daisy Chain）方式连接。
    *   **信号路径**：主机 **GPIO15** 连接至第一颗 LED 的 **DIN** 引脚。随后的每一级 LED 的 **DOUT** 连接至下一级的 **DIN**。
*   **供电**：LED 灯珠的 VCC 引脚连接至 **5V** 总线，而非 3.3V，以确保亮度和色彩稳定性。

    *   信号直接路由至 **GPIO34**。由于 GPIO34 是 ESP32 的 ADC1 通道输入引脚，主机通过模数转换读取音频波形数据。
*   **电路细节**：包含必要的偏置电阻和滤波电容，以确保模拟信号的信噪比。

### 4. 红外发射电路 (IR TX)
*   **功能**：实现红外遥控信号发射。
*   **驱动逻辑**：
    *   控制信号由 **GPIO13** 提供。
    *   由于 GPIO 驱动能力有限，电路采用晶体管（BJT 或 MOSFET）作为开关元件构建驱动电路。
    *   **GPIO13** 高电平导通晶体管，从而驱动红外发射二极管（IR LED）导通发光。

### 5. 扩展接口 (HY2.0-4P)
底座侧面引出了两个 Grove 兼容接口（HY2.0-4P 插座），分别对应不同的功能定义：

    *   **G36**：ESP32 的 Sensor_VP 引脚，仅支持作为模拟/数字输入（Input Only），不可作为输出使用。
*   **PORT.C (UART)**：
    *   **引脚定义**：GND, 5V, **G17**, **G16**。
    *   **G17**：连接至 UART TXD (U2TXD)。
    *   **G16**：连接至 UART RXD (U2RXD)。
    *   此接口常用于连接串口通信设备或传感器。

### 6. Pogo Pin 与充电接口

*   **充电功能**：包含 5V 输入和 GND 触点。当放置在充电座上时，外部 5V 通过触点进入底座，进而通过 M5-Bus 输送给主机的电源管理芯片（PMIC）进行电池充电。
*   **通讯功能**：引出了 I2C 总线。
    *   **SDA**：连接至 **GPIO21**。
    *   **SCL**：连接至 **GPIO22**。
    *   允许充电底座或其他磁吸附件与主机进行 I2C 通信。

    *   充电管理（恒流/恒压控制）和 5V/3.3V 稳压均依赖连接的 M5Stack Core 主机内部电路完成。
