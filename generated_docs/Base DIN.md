Base DIN 是一款专为 M5Core 系列主机设计的 DIN 导轨底座扩展板。其电路架构主要围绕宽电压输入电源管理、锂电池充放电控制、M5-Bus 总线扩展以及外设接口映射（Port B/C）展开。该设计旨在为 Core 主机提供工业导轨安装能力、独立的供电系统以及灵活的信号引出接口。

*   **DC-DC 降压电路 (SY8303AIC)**

    *   **储能与稳压**：芯片配合功率电感（L）和输出电容组成 Buck 拓扑电路，将高压直流输入转换为稳定的 5V 系统电压。
    *   **输出能力**：该电路为 M5-Bus 的 5V 电压轨提供主供电，驱动 Core 主机及外设。

*   **锂电池充电管理 (TP4057)**
    *   **充电芯片**：使用 **TP4057** 线性锂电池充电管理芯片。
    *   **充电逻辑**：当外部电源（5V）存在时，芯片对板载 500mAh 锂聚合物电池进行恒流/恒压（CC/CV）充电。
    *   **电流配置**：通过 PROG 管脚的对地电阻设定充电电流。
    *   **状态指示**：芯片的 CHRG 和 STDBY 管脚连接有 LED 指示灯，分别用于指示“正在充电”和“充电完成/待机”状态。

*   **电源切换与物理控制**

    *   **切断逻辑**：该开关串联在电源路径的关键节点，能够物理切断电池对系统的供电回路，以及外部 DC 输入对系统的供电，确保在关闭状态下设备完全断电，防止电池过放。

    *   **BAT (Pin 30)**：电池正极直接连接点（经开关），允许 M5Core 读取电池电压以监测电量。
