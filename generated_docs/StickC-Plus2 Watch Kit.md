## 原理图分析

以下是基于 M5Stack StickC-Plus2 硬件规格及原理图逻辑的结构化技术描述：

### 1. 核心处理器与存储系统
*   **主控 SoC**：核心采用 **ESP32-PICO-V3-02** 系统级封装芯片（SiP）。该芯片内部集成 ESP32 双核 Xtensa® 32-bit LX6 微处理器、晶振、滤波电容、**8MB SPI Flash** 以及 **2MB PSRAM**。
*   **射频电路**：SoC 的 LNA_IN 引脚连接至由电感和电容组成的 π 型匹配网络，随后连接至板载 **2.4GHz 3D 天线**，用于 Wi-Fi 和 Bluetooth 通讯，确保无线信号的阻抗匹配与辐射效率。
*   **外围配置**：包含标准的去耦电容网络，以保证内核电压（VDD33, VDD_SDIO 等）的稳定性。

### 2. 电源管理与开关逻辑
*   **电源输入与充电**：
    *   **VBUS**：来自 Type-C 接口的 5V 输入，经过电源管理电路为系统供电并为电池充电。
    *   **VBAT**：连接 200mAh 聚合物锂电池。电池电压经过电压调整电路（LDO 或 DC-DC）转换为系统所需的 3.3V。
*   **电源保持与开关机电路（Power Hold）**：
    *   **开机触发**：通过 **Button C（按键 C）** 按下或 RTC 芯片（BM8563）的 **IRQ 中断信号** 拉低，导通电源使能端，启动 3.3V 稳压器输出。
    *   **电源维持（Hold）**：ESP32 启动后，通过 **GPIO4** 输出高电平。该信号控制电源锁存电路（通常为三极管或 MOSFET 组合），在 Button C 释放后继续维持稳压器的使能状态，实现软件控制的电源保持。
    *   **关机逻辑**：当 ESP32 将 GPIO4 置为低电平（或复位导致引脚浮空）时，锁存电路断开；若此时 Button C 未按下，3.3V 输出切断，系统断电。

### 3. 显示与音频子系统
*   **LCD 显示屏**：
    *   采用 **ST7789v2** 驱动芯片的 1.14 英寸 TFT LCD。
    *   **接口**：使用 4-wire SPI 接口连接。
    *   **引脚映射**：SCLK 连接 GPIO13，MOSI 连接 GPIO15，CS（片选）连接 GPIO5，DC（数据/命令）连接 GPIO14，RST（复位）连接 GPIO12。
    *   **背光控制**：背光负极通过 MOSFET 受控于 **GPIO27**，支持 PWM 调光。
*   **音频电路**：
    *   **数字麦克风**：板载 **SPM1423** PDM 麦克风。CLK 时钟线连接 **GPIO0**，DATA 数据线连接 **GPIO34**。
    *   **蜂鸣器**：无源蜂鸣器通过驱动晶体管连接至 **GPIO2**，由 ESP32 输出 PWM 信号驱动发声。

### 4. 传感器与实时时钟
*   **总线架构**：板载传感器挂载于内部 **I2C 总线**，SCL 连接 **GPIO22**，SDA 连接 **GPIO21**。
*   **惯性测量单元 (IMU)**：采用 **MPU6886** 6 轴运动传感器（3 轴陀螺仪 + 3 轴加速度计），连接至 I2C 总线，用于姿态检测与运动追踪。
*   **实时时钟 (RTC)**：采用 **BM8563** 芯片，连接至 I2C 总线。其 **INT（中断）引脚** 连接至电源管理电路的开机触发端，支持定时唤醒系统功能。RTC 具有独立的备份供电路径（或依托主电池微电流维持），保证系统断电后时间不丢失。

### 5. 接口与通讯
*   **USB 转串口**：
    *   采用 **CH9102**（或同类）USB 转 UART 芯片。
    *   **数据路径**：USB 端连接 Type-C 的 D+/D- 差分线；UART 端连接 ESP32 的 **GPIO19 (TXD)** 和 **GPIO20 (RXD)** 进行全双工通讯。
    *   **自动下载电路**：DTR 和 RTS 信号通过双三极管逻辑电路控制 ESP32 的 EN 和 IO0 引脚，实现固件自动烧录模式的进入。
*   **Grove 接口 (HY2.0-4P)**：
    *   提供 VDD (5V)、GND 以及两路信号线。信号线连接至 **GPIO32** 和 **GPIO33**，可配置为 I2C、UART 或普通 GPIO 使用。
*   **外部排针扩展**：
    *   顶部与底部排针引出了以下信号：**G0, G25, G26**（支持 DAC/ADC）, **G36**（仅输入，VP）, **G32, G33**。同时提供 5V, 3.3V, GND 和 BAT 电源引脚。

### 6. 逻辑与交互
*   **按键输入**：
    *   **Button A (M5 键)**：正面大按键，连接 **GPIO37**，低电平有效。
    *   **Button B (侧键)**：连接 **GPIO39**，低电平有效。
    *   **Button C (电源/自定义键)**：除具备硬件电源唤醒功能外，还连接至 **GPIO35**，系统启动后可作为普通输入按键使用。
*   **指示灯**：
    *   **电源指示 LED**：绿色 LED。该 LED 物理连接至 3.3V 电源轨（通常串联限流电阻），用于指示系统 3.3V 供电是否正常，**不可通过 GPIO 编程控制**。
