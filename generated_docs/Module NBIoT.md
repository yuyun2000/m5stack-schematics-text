*   **供电输入**：模组的 `VBAT` 引脚连接至经过滤波的主电源轨，旁边并联了多个去耦电容（通常包括 100uF 钽电容及 100nF, 33pF 陶瓷电容），以应对发射时的瞬态电流需求并滤除高频噪声。

### 2. 电源管理与开关机逻辑
*   **主供电路径**：模块通过 M5-Bus 的 3.3V 引脚获取电源。
*   **开关机控制 (PWRKEY)**：
    *   电路设计了一个由 **GPIO2** 控制的开关机电路。
    *   **开机逻辑**：`PWRKEY` 引脚通常需被拉低以触发开机。电路中包含一个三极管或逻辑驱动电路，当主控端的 **GPIO2** 输出高电平时，驱动电路导通，将 M5311 的 `PWRKEY` 拉至有效电平。
    *   **物理按键并联**：板载物理电源按钮与 GPIO2 控制路径并联。按下按钮可直接将 `PWRKEY` 信号接地（或拉至有效电平），实现手动开关机。
    *   **时序控制**：硬件电路支持软件定义的时序，即 GPIO2 维持高电平 2秒以上触发开机，维持 8秒以上触发关机。
*   **复位电路 (RESET)**：
    *   模组的 `RST` 引脚连接至 M5-Bus 的 **GPIO26**。
    *   主控可通过拉低或翻转 GPIO26 的电平来实现模组的硬件复位，用于在死机或网络异常时重启模组。
*   **低功耗设计**：电路支持 PSM (Power Saving Mode) 模式。在 PSM 模式下，模组耗流可低至 3uA。电源网络设计确保在模组进入休眠状态时，外围非必要电路（如状态 LED）不会产生漏电流。

    *   **配置方式**：用户可以通过割断默认的 GPIO16/17 连线，并焊接飞线或电阻至备用引脚焊盘，将 UART 信号重定向至未冲突的 GPIO。

### 4. 射频前端与天线电路
*   **射频路径**：M5311 的 `ANT` 引脚输出后，经过一组 π 型 (Pi-network) 阻抗匹配电路（由电感和电容组成），用于优化 50Ω 阻抗匹配。
